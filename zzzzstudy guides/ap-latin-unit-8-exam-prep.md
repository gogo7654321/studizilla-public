### Before Translating

```
READ THE PASSAGE:
- Get overall sense
- Note familiar vocabulary
- Identify challenging spots
- Observe punctuation

IDENTIFY STRUCTURES:
- Main clauses
- Subordinate clauses
- Ablative absolutes
- Participle phrases

MARK KEY ELEMENTS:
- Subjects
- Main verbs
- Relative pronouns
- Subordinating conjunctions
```

### Translation Process

```
STEP 1: FIND MAIN VERB(S)
What's the backbone?

STEP 2: FIND SUBJECT(S)
Who/what does the action?

STEP 3: BRACKET SUBORDINATES
[Relative clauses]
[Cum clauses]
[Ablative absolutes]

STEP 4: TRANSLATE MAIN CLAUSE
Get the skeleton

STEP 5: ADD SUBORDINATES
Layer in complexity

STEP 6: SMOOTH ENGLISH
Make it readable
Maintain accuracy
```

### Common Errors to Avoid

```
1. WRONG CASE IDENTIFICATION:
   "militum" = of soldiers (genitive)
   NOT "soldiers" (nominative)

2. PARTICIPLE ERRORS:
   Perfect passive ‚â† Present active
   "victus" = having been conquered
   NOT "conquering"

3. TENSE CONFUSION:
   Latin perfect can = English past OR present perfect
   Check context

4. WORD ORDER TRANSFER:
   Latin word order ‚â† English
   Reorder for English sense

5. OMISSIONS:
   Every word matters
   Check you've rendered all

6. ADDITIONS:
   Don't add words not there
   Interpretation ‚â† translation
```

### Vergil vs. Caesar Translation

```
CAESAR CHALLENGES:
- Long sentences
- Indirect statement chains
- Multiple ablative absolutes
- Military vocabulary
- Pronoun reference

CAESAR HELPS:
- Straightforward word order
- Predictable structures
- Clear logical connections
- Limited metaphor


VERGIL CHALLENGES:
- Poetic word order
- Separated adjective-noun pairs
- Elision
- Figurative language
- Mythological references

VERGIL HELPS:
- Meter constrains options
- Emotional vocabulary often clear
- Epic formulas recognizable
- Syllabus passages familiar
```

## üìù Sub-Theme 4: Essay Strategies

### Analytical Essay Overview

```
TASK:
Analyze how an author uses literary
devices/techniques to achieve an effect

TIME:
~45 minutes recommended

STRUCTURE:
- Introduction with thesis
- 2-3 body paragraphs
- Conclusion

REQUIREMENTS:
- Clear thesis
- Latin evidence (quotes)
- Accurate translation
- Literary analysis
- Coherent argument
```

### Building Strong Thesis

```
WEAK THESIS:
"Vergil uses literary devices."
"Caesar writes about war."

MODERATE THESIS:
"Vergil uses imagery to create emotion."
"Caesar uses indirect statement to
present enemy perspectives."

STRONG THESIS:
"Through the strategic juxtaposition
of Dido's passionate diction with
Aeneas's restrained responses, Vergil
dramatizes the conflict between personal
desire and public duty, ultimately
suggesting that pietas demands the
sacrifice of eros."
```

### Using Latin Evidence

```
FORMAT:

1. CLAIM:
   State your analytical point

2. EVIDENCE:
   Quote Latin (with line reference)

3. TRANSLATION:
   Render accurately

4. ANALYSIS:
   Explain how it supports claim

EXAMPLE:

CLAIM: "Vergil emphasizes Dido's suffering
through disease imagery."

EVIDENCE: "At regina gravi iamdudum
saucia cura / vulnus alit venis et
caeco carpitur igni" (4.1-2)

TRANSLATION: "But the queen, long since
wounded by heavy care, nourishes the
wound in her veins and is seized by
hidden fire."

ANALYSIS: "The progression from 'saucia'
(wounded) to 'vulnus' (wound) to 'carpitur'
(is seized/consumed) creates an image of
love as disease, consuming Dido from within.
The 'hidden fire' (caeco...igni) suggests
both passion and destruction, foreshadowing
her eventual immolation."
```

### Common Essay Mistakes

```
AVOID:

1. PLOT SUMMARY:
   Telling what happens ‚â† analysis
   Always explain WHY and HOW

2. FEATURE SPOTTING:
   "There is alliteration" isn't enough
   Must explain its EFFECT

3. TRANSLATION ERRORS:
   Undermines credibility
   Verify before writing

4. NO THESIS:
   Collection of observations ‚â† essay
   Need unified argument

5. IGNORING PROMPT:
   Answer what's asked
   Address all parts

6. INSUFFICIENT EVIDENCE:
   One quote isn't enough
   Multiple examples strengthen
```

### Grammar Terms

```
MUST KNOW:

ABLATIVE ABSOLUTE:
Noun + participle in ablative
"His rebus cognitis..."

INDIRECT STATEMENT:
Accusative + infinitive
"Dixit se venisse..."

RELATIVE CLAUSE OF PURPOSE:
Relative pronoun + subjunctive
"Legatos misit qui pacem peterent"

RESULT CLAUSE:
Ut/ut non + subjunctive
"Tam fortis erat ut vinceret"

Ut/ne + subjunctive
"Venit ut videret"

CUM CLAUSE:
Cum + subjunctive (circumstantial)
"Cum haec dixisset..."

GERUNDIVE:
Passive periphrastic
"Haec facienda sunt"
```

## üìè Sub-Theme 6: Scansion Review

### Quick Reference

```
DACTYLIC HEXAMETER:
6 feet per line
Final foot: ‚Äî x (spondee/trochee)
5th foot: almost always dactyl ‚Äî ‚à™ ‚à™
Feet 1-4: dactyl OR spondee

DACTYL: ‚Äî ‚à™ ‚à™ (long-short-short)
SPONDEE: ‚Äî ‚Äî (long-long)

LONG VOWELS:
- Natural length (macron)
- Before two consonants
- Diphthongs (ae, au, oe, etc.)

ELISION:
Final vowel/m + initial vowel/h
= final is not pronounced
```

### Scansion Steps

```
1. MARK ELISIONS:
   Word-final vowel/m before word-initial vowel
   Put in parentheses

2. MARK KNOWN LONGS:
   Diphthongs
   Before double consonants
   Known long vowels

3. MARK LAST TWO FEET:
   5th = usually dactyl ‚Äî ‚à™ ‚à™
   6th = ‚Äî x

4. WORK BACKWARDS:
   Fill in remaining feet
   Each must be ‚Äî ‚à™ ‚à™ or ‚Äî ‚Äî

5. VERIFY:
   Total of 6 feet
   All patterns valid
   Makes sense metrically
```

### Common Patterns

```
DSSS (Dactyl-Spondee-Spondee-Spondee):
Slow, heavy, solemn

DDDD (All dactyls):
Fast, light, energetic

DSDS (Alternating):
Common, balanced

SPONDAIC LINE (rare):
Spondee in 5th foot
Special effect

GOLDEN LINE:
Adjective A - Adjective B - Verb - Noun A - Noun B
"aurea purpuream subnectit fibula vestem"
```
