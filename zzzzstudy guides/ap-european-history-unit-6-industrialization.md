## ÔøΩüìã Unit Overview

The Industrial Revolution transformed Europe more profoundly than any event since the Agricultural Revolution. Beginning in Britain, industrialization spread across the continent, creating new forms of production, new social classes, new ideologies, and new problems. Cities exploded in size, working conditions were often horrific, but eventually material conditions improved for many. The modern world was being born! üè≠

## üá¨üáß Industrial Revolution in Britain

### Why Britain First?

| Factor | Description |
|--------|-------------|
| **Geography** | Coal and iron deposits; rivers; ports |
| **Agriculture** | Agricultural Revolution; surplus labor |

| **Culture** | Entrepreneurship; science valued |

### Agricultural Revolution

| Innovation | Impact |
|------------|--------|
| **Enclosure** | Consolidated land; efficient but displaced |
| **Crop rotation** | Improved yields |
| **Selective breeding** | Better livestock |
| **New crops** | Turnips, clover |

## üåç Spread of Industrialization

### Pattern of Spread

| Region | Timing | Features |
|--------|--------|----------|
| **Britain** | 1760s+ | First; led for decades |
| **Belgium** | 1820s | First continental; coal, iron |
| **France** | 1830s+ | Slower; luxury goods |
| **Germany** | 1850s+ | Rapid; state-led; steel |
| **United States** | 1840s+ | Resources; immigration |
| **Russia** | 1890s+ | Late; state-driven |

### Why Slower Elsewhere?

| Factor | Description |
|--------|-------------|
| **Resources** | Not all had coal/iron |
| **Capital** | Less available |
| **Skills** | Needed training |
| **British lead** | Competition difficult |
| **Politics** | Wars; instability |
| **Agriculture** | Still feudal in some areas |

### Government Role

| Country | Approach |
|---------|----------|
| **Britain** | Laissez-faire; private |
| **Germany** | State support; tariffs; Zollverein |
| **France** | State investment; slower |
| **Russia** | State-directed; Witte |
| **Japan** | Meiji government-led |

### German Industrialization

|---------|-------------|

| **Railroads** | State-supported |
| **Banks** | Industrial investment |

| **Late advantage** | Newest technology |

## üë∑ Working and Living Conditions

### Factory Conditions

| Condition | Description |
|-----------|-------------|
| **Hours** | 14-16 hours/day, 6 days/week |
| **Wages** | Low; often in company tokens |
| **Safety** | Machines dangerous; no guards |
| **Discipline** | Fines; beatings |
| **Child labor** | Young children in factories, mines |
| **Women** | Lower wages than men |

### Child Labor

| Aspect | Details |
|--------|---------|
| **Ages** | As young as 5-6 |
| **Jobs** | Textile mills; coal mines; climbing boys |
| **Hours** | Same as adults |
| **Punishment** | Physical abuse common |
| **Wages** | Fraction of adult wages |

### Mining Conditions

| Condition | Description |
|-----------|-------------|
| **Danger** | Explosions; cave-ins; flooding |
| **Darkness** | Work by candlelight |
| **Children** | Opened doors; pulled carts |
| **Women** | Worked underground (until banned) |
| **Health** | Lung disease; injuries |

### Factory Reform (Britain)

| Act | Year | Provisions |
|-----|------|------------|

| **Mines Act** | 1842 | No women/children underground |

| **Later acts** | 1870s+ | Gradually expanded protections |

## üí≠ New Ideologies

### Conservatism

| Thinker | Ideas |
|---------|-------|
| **Edmund Burke** | Tradition; gradual change; distrust of revolution |
| **Joseph de Maistre** | Restore throne and altar |
| **Core beliefs** | Order; hierarchy; religion; tradition |
| **After 1815** | Dominant in Concert of Europe |

### Liberalism

| Principle | Description |
|-----------|-------------|
| **Individual rights** | Liberty; property |
| **Limited government** | Constitutional; rule of law |
| **Free market** | Laissez-faire economics |
| **Meritocracy** | Careers open to talent |
| **Suffrage** | Property owners (not universal) |
| **Progress** | Society can improve |

### Classical Liberalism - Key Thinkers

| Thinker | Contribution |
|---------|--------------|
| **Adam Smith** | Free market economics |
| **Jeremy Bentham** | Utilitarianism; "greatest good for greatest number" |
| **John Stuart Mill** | Liberty; later modified laissez-faire; women's rights |
| **David Ricardo** | Iron law of wages; comparative advantage |
| **Thomas Malthus** | Population outpaces food |

### Utilitarianism

| Concept | Description |
|---------|-------------|
| **Principle** | Greatest good for greatest number |
| **Bentham** | Calculate pleasure/pain |
| **Mill** | Qualitative distinctions; liberty |
| **Impact** | Justified reform; influenced law |

### Nationalism

| Element | Description |
|---------|-------------|
| **Nation** | Group with shared identity |
| **State** | Political unit |
| **Goal** | Each nation should have own state |
| **Basis** | Language; culture; history; ethnicity |
| **Types** | Unification; independence |

### Types of Nationalism

| Type | Examples |
|------|----------|

| **Separatist** | Greece; Belgium; Poland |
| **Reform** | Within existing states |
| **Later** | Aggressive; imperialist |

### Early Socialism
