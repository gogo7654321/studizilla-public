## üîç What Is Synthesis?

### Definition

| Synthesis | The combination of ideas from multiple sources into a new, unified argument |
|-----------|---------------------------------------------------------------------------|

### Synthesis vs. Summary

| Summary | Synthesis |
|---------|-----------|
| Describes what one source says | Combines multiple sources |
| Source-by-source organization | Idea-by-idea organization |
| Reporter role | Conversation facilitator role |
| "Source A says... Source B says..." | "Both sources suggest... However, they differ on..." |
| Passive | Active, creative |

### Synthesis vs. Comparison

| Comparison | Synthesis |
|------------|-----------|
| Points out similarities and differences | Uses sources to build YOUR argument |
| May or may not take a position | Always advances YOUR thesis |
| About the sources | About your ideas |

### The Goal of Synthesis

| Goal | Description |
|------|-------------|
| **Create something new** | Not just reporting what sources say |
| **Build your argument** | Sources support YOUR thesis |
| **Show conversation** | How sources relate to each other |
| **Demonstrate understanding** | You grasp the complexity of the issue |

## üìö Understanding Sources

### Source Types

| Type | Characteristics | Best For |
|------|-----------------|----------|
| **Scholarly article** | Peer-reviewed, specialized | Academic claims |
| **News article** | Current, accessible | Recent events |
| **Government report** | Official data, comprehensive | Statistics, policy |
| **Essay/Op-ed** | Opinion, persuasive | Perspective |
| **Study/Research** | Data, methodology | Evidence |
| **Interview** | Primary, personal | Humanizing |
| **Visual** | Chart, graph, image | Data, illustration |

### Source Evaluation: CRAAP Test

| Criterion | Questions |
|-----------|-----------|
| **Currency** | When was it published? Is it current? |
| **Relevance** | Does it relate to your argument? |
| **Authority** | Who created it? What credentials? |
| **Accuracy** | Is it supported? Verified? |
| **Purpose** | Why was it created? What bias? |

### Source Relationships

| Relationship | Description | Signal Phrases |
|--------------|-------------|----------------|
| **Agreement** | Sources support same point | "Both X and Y argue..." |
| **Disagreement** | Sources conflict | "While X claims..., Y contends..." |
| **Qualification** | One limits another | "Although X is generally true, Y notes..." |
| **Extension** | One builds on another | "X's research is extended by Y's finding that..." |
| **Complementary** | Different angles on same issue | "X examines the economic angle, while Y considers the social impact" |

## ‚úèÔ∏è Integrating Sources

### Integration Methods

| Method | When to Use | Example |
|--------|-------------|---------|
| **Direct quote** | Language is significant | Smith argues, "exact words" (Source A). |
| **Paraphrase** | Ideas matter more than words | Smith contends that idea restated (Source A). |
| **Summary** | Broad overview needed | Smith's research examines topic (Source A). |

### Signal Phrases

| Purpose | Examples |
|---------|----------|
| **Neutral** | notes, states, observes, reports, describes |
| **Strong claim** | argues, claims, asserts, contends, maintains |
| **Agreement** | confirms, supports, validates, echoes |
| **Disagreement** | disputes, challenges, questions, refutes |
| **Suggestion** | implies, suggests, indicates, hints |

### Signal Phrase Variety

| Pattern | Example |
|---------|---------|
| **Author + verb** | Smith argues that... |
| **According to** | According to Source A, ... |
| **Source description + verb** | A recent study shows... |
| **Mid-sentence** | The problem, as Smith notes, is... |
| **End citation** | Evidence suggests this trend (Source B). |

### Embedding Quotes

| Poor | Strong |
|------|--------|
| "Education is key." | As education researcher Smith notes, "education is key" to economic mobility (Source A). |
| Source A says, "Education is key." | The data confirms that "education is key," particularly for underserved populations (Source A). |

### Quote Sandwich

| Layer | Function | Example |
|-------|----------|---------|
| **Introduction** | Context, signal phrase | Research supports this connection. |
| **Quote** | Evidence | As the CDC reports, "screen time correlates with anxiety" (Source C). |
| **Explanation** | Commentary | This data suggests that limiting device use could improve mental health outcomes. |

## üìù The AP Synthesis Essay

### Essay Requirements

| Requirement | Details |
|-------------|---------|
| **Thesis** | Take a clear position on the prompt |
| **Sources** | Integrate at least 3 of the provided sources |
| **Your argument** | Sources support YOUR thesis, not just report what they say |
| **Citation** | Reference sources by letter or author |

### Essay Structure

| Section | Content |
|---------|---------|
| **Introduction** | Context + thesis taking position on prompt |
| **Body 1** | First supporting point + evidence from 2+ sources + commentary |
| **Body 2** | Second supporting point + evidence from 2+ sources + commentary |
| **Body 3** | Third point OR counterargument + evidence + response |
| **Conclusion** | Synthesis of ideas + broader implications |

### Sample Thesis Structures

| Structure | Example |
|-----------|---------|
| **Position + reasons** | "Schools should start later because research shows improved academic performance, better mental health, and reduced accidents." |
| **Concession + position** | "Although some argue scheduling is difficult, schools should start later because the academic benefits outweigh logistical challenges." |
| **Acknowledgment + position** | "While there are valid concerns about extracurriculars, later start times remain essential for student wellbeing." |

### Body Paragraph Structure

| Component | Function |
|-----------|----------|
| **Topic sentence** | Claim for this paragraph (supports thesis) |
| **Source 1 evidence** | Integrated quote/paraphrase |
| **Commentary** | Explain significance |
| **Source 2 evidence** | Additional support or contrast |
| **Commentary** | Connect to thesis |
| **Transition** | Move to next point |

## üîÑ Organizing by Idea

### Source-by-Source (AVOID)

```
Paragraph 1: Source A says...
Paragraph 2: Source B says...
Paragraph 3: Source C says...
```

**Problem:** This is summary, not synthesis!

### Idea-by-Idea (USE THIS)

```
Paragraph 1: [Idea 1]
- Evidence from Source A
- Evidence from Source C

Paragraph 2: [Idea 2]
- Evidence from Source B
- Evidence from Source D

Paragraph 3: [Counterargument/Idea 3]
- Evidence from Source E
- Response using Source A
```

**Benefit:** This is true synthesis!

### Synthesis Matrix

| Idea/Theme | Source A | Source B | Source C |
|------------|----------|----------|----------|
| Academic impact | ‚úì supports | - | ‚úì supports |
| Economic effects | ‚úì concerns | ‚úì supports | - |
| Social implications | - | ‚úì concerns | ‚úì supports |

Use this matrix to plan paragraphs organized by idea, not source!

## üé® Advanced Synthesis Techniques

### Source Conversation Moves

| Move | Description | Template |
|------|-------------|----------|
| **Agreement synthesis** | Sources reinforce each other | "Both Source A and Source C confirm that..." |
| **Disagreement synthesis** | Sources contradict | "While Source A argues X, Source B contends Y, revealing..." |
| **Extension synthesis** | One source builds on another | "Source D extends Source A's findings by demonstrating..." |
| **Complication synthesis** | One qualifies another | "Source C complicates Source A's claim by showing..." |
| **Gap synthesis** | What sources don't address | "What neither source considers is..." |

### Strategic Source Selection

| Strategy | When to Use |
|----------|-------------|
| **Strongest support** | Your most compelling point |
| **Expert authority** | Establishing credibility |
| **Data/statistics** | Objective evidence needed |
| **Personal narrative** | Humanizing the issue |
| **Counterargument** | Addressing opposition fairly |
| **Visual/chart** | Illustrating patterns |

### Balancing Your Voice and Sources

| Problem | Solution |
|---------|----------|
| **Sources overwhelm** | Lead with your ideas; sources support |
| **Too little sourcing** | Add more evidence; weave in more voices |
| **Choppy integration** | Vary signal phrases; use sandwich method |
| **No conversation** | Show how sources relate to each other |

## üìñ Extended Synthesis Analysis Examples

### Example 1: Agreement Synthesis

**Your Claim:** School start times significantly impact student health.

**Source A:** CDC study showing teen sleep deprivation correlates with later school start times.

**Source C:** American Academy of Pediatrics recommendation for 8:30 AM or later starts.

**Synthesized Paragraph:**
> The scientific consensus supporting later school start times is overwhelming. According to the CDC, adolescents who start school before 8:30 AM are significantly more likely to experience chronic sleep deprivation (Source A). This finding is reinforced by the American Academy of Pediatrics, which officially recommends that middle and high schools begin at 8:30 AM or later (Source C). Taken together, these sources establish that the current practice of early start times contradicts established health science, making schedule reform not merely desirable but medically necessary.

| Element | What It Does |
|---------|--------------|
| **Topic sentence** | States YOUR claim |
| **Source A** | First evidence |
| **Source C** | Reinforcing evidence |
| **Synthesis** | "Taken together" connects sources |
| **Commentary** | Explains significance |

### Example 2: Disagreement Synthesis

**Your Claim:** The economic argument against later start times is overstated.

**Source B:** School district report citing transportation costs of schedule change.

**Source E:** Study showing long-term economic benefits of better student outcomes.

**Synthesized Paragraph:**
> While economic concerns about later start times are understandable, they rely on short-term thinking. Source B emphasizes the immediate costs of transportation adjustments, including additional bus routes and driver wages. However, Source E's longitudinal study reveals that these upfront costs are dwarfed by long-term benefits: improved graduation rates, higher lifetime earnings, and reduced healthcare costs. By focusing only on immediate expenditures, opponents overlook the substantial return on investment that improved student health generates over time.

| Element | What It Does |
|---------|--------------|
| **Concession** | Acknowledges opposition fairly |
| **Source B** | Counterargument evidence |
| **Source E** | Your supporting evidence |
| **Synthesis** | Shows how sources relate |
| **Commentary** | Explains why your position is stronger |

### Example 3: Complication Synthesis

**Your Claim:** The solution requires considering multiple stakeholders.

**Source A:** Focus on student health benefits.

**Source D:** Parent concerns about afternoon supervision.

**Synthesized Paragraph:**
> Any viable proposal must balance competing needs. Source A's compelling data on student health benefits must be weighed against the practical concerns raised in Source D, where parents express worry about unsupervised afternoon hours. However, this apparent conflict dissolves when we recognize that both sources share a fundamental concern: student welfare. Rather than treating these as irreconcilable, schools should develop solutions that address both‚Äîperhaps through after-school programming that extends supervision while still allowing later start times.

| Element | What It Does |
|---------|--------------|
| **Framing** | Establishes complexity |
| **Source A** | Health perspective |
| **Source D** | Parent perspective |
| **Synthesis** | Finds common ground |
| **Solution** | Your synthesized position |

## üéØ Common Synthesis Essay Pitfalls

- Brief context on the debate
- Thesis with clear position

**Body Paragraphs (70-75%):**

*Pattern A: Agreement + Extension*
- Topic sentence (your claim)
- Source 1 evidence + commentary
- Source 2 evidence (extends first) + commentary
- Synthesis statement

*Pattern B: Concession + Rebuttal*
- Acknowledge opposition (Source X)
- Concede what's valid
- Counter with your sources
- Show why your position is stronger

**Conclusion (10-12%):**
- Synthesize main points (not repeat)
- Broader implications
- Call to action or forward-looking statement
