## ‚ùì 4.1 Developing Your Question

### Choosing a Topic

```
TOPIC SELECTION:

PERSONAL INTEREST:
What genuinely interests you?
What would you enjoy researching?
What do you wonder about?

RESEARCHABILITY:
Are there quality sources?
Is there scholarly discussion?
Can you access materials?

COMPLEXITY:
Are there multiple perspectives?
Is there genuine debate?
Can you add something?

SCOPE:
Narrow enough to address fully
Broad enough for 2,000 words
Appropriate for 6+ sources
```

### Question Development

```
REFINING YOUR QUESTION:

START BROAD:
"Artificial intelligence"

ADD FOCUS:
"AI in education"

ADD COMPLEXITY:
"Should AI be used in K-12 classrooms?"

MAKE ARGUABLE:
"To what extent should AI-powered 
tools be integrated into K-12 
education, and what safeguards 
are necessary?"

TEST YOUR QUESTION:
‚ñ° Debatable (not yes/no)?
‚ñ° Complex (multiple perspectives)?
‚ñ° Researchable (sources exist)?
‚ñ° Significant (matters)?
‚ñ° Focused (can address fully)?
```

### Research Planning

```
RESEARCH PLAN:

1. PRELIMINARY EXPLORATION
   - Background reading
   - Identify key debates
   - Find major sources
   - Refine question

2. SYSTEMATIC RESEARCH
   - Academic databases
   - Multiple source types
   - Multiple perspectives
   - Primary sources if possible

3. SOURCE TRACKING
   - Full citations from start
   - Notes with page numbers
   - Connections between sources
   - Gaps to fill

4. SYNTHESIS DEVELOPMENT
   - Themes emerge
   - Position develops
   - Argument forms
   - Thesis clarifies
```

```

‚îú‚îÄ‚îÄ Question/thesis preview

### Multimedia Design

```

- Consistent template
- Readable fonts

- Low-quality images

## üó£Ô∏è 4.4 Oral Defense

### Defense Overview

```
ORAL DEFENSE:

FORMAT:
- 2-3 questions
- Following presentation
- Individual response
- Tests understanding

QUESTION TYPES:
‚îú‚îÄ‚îÄ Clarification
‚îÇ   "Can you explain...?"
‚îÇ
‚îú‚îÄ‚îÄ Extension
‚îÇ   "How might this apply to...?"
‚îÇ
‚îú‚îÄ‚îÄ Challenge
‚îÇ   "What about the argument that...?"
‚îÇ
‚îî‚îÄ‚îÄ Reflection
    "Why did you...?"
```

### Preparation Strategies

```
PREPARING FOR DEFENSE:

KNOW YOUR SOURCES:
- Read beyond what you cited
- Understand context
- Know author backgrounds
- Be ready for questions

ANTICIPATE QUESTIONS:
- What's unclear?
- What's controversial?
- What needs more evidence?
- What are the limitations?

PREPARE ANSWERS:
- Draft key responses
- Practice articulating
- Know supporting evidence
- Have examples ready

PRACTICE Q&A:
- Have someone quiz you
- Practice thinking quickly
- Time your responses
- Get comfortable
```

### Response Techniques

```
ANSWERING QUESTIONS:

STRUCTURE:
1. Acknowledge (thank/recognize)
2. Clarify (if needed)
3. Answer directly
4. Support (evidence/example)
5. Connect (to argument)

PHRASES:
"That's a great question..."
"To address that point..."
"The research suggests..."
"Building on what I argued..."
"To clarify my position..."

IF YOU DON'T KNOW:
"That's beyond my research scope,
but based on what I found..."
"I'd need to investigate further,
but I would hypothesize..."

STAY COMPOSED:
- Pause before answering
- Think clearly
- Speak confidently
- Acknowledge limits honestly
```

### Defense Scoring

```
ROW 8: ORAL DEFENSE (6 pts)

HIGH SCORE (5-6):
- Thoroughly addresses questions
- Defends with evidence
- Extends understanding
- Acknowledges limitations
- Shows deep knowledge

MEDIUM SCORE (3-4):
- Adequately addresses questions
- Some support provided
- Basic understanding shown
- Some limitations noted

LOW SCORE (1-2):
- Limited response
- Minimal support
- Surface understanding
- Avoids or can't answer
```

```
DRAFTING STRATEGIES:

- Structure your argument
- Plan evidence placement
- Note gaps to fill
- Organize logically

DRAFT MESSILY:
- Get ideas down
- Don't edit while drafting
- Write through problems
- Mark areas to return to

REVISE SYSTEMATICALLY:
Pass 1: Argument and structure
Pass 2: Evidence and analysis
Pass 3: Style and clarity
Pass 4: Citations and format

GET FEEDBACK:
- Peer review
- Teacher conference
- Read aloud

```

- Note key arguments
- Identify perspectives
- Assess credibility

COMMON TASKS:
- Identify main argument
- Evaluate evidence
- Compare perspectives
- Assess credibility
- Explain reasoning

FORMAT:
- Direct answers
- Brief but complete
- Evidence from text
- Clear reasoning
```

```
PART B: SYNTHESIS ESSAY

- Read all sources first
- Identify themes
- Plan before writing
- Synthesize, don't summarize

STRUCTURE:
- Clear thesis
- Synthesis of sources
- Analysis of evidence

- Strong conclusion

TIME MANAGEMENT:
- 15 min: Read and plan
- 60 min: Write
- 15 min: Revise
