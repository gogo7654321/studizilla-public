## üì∏ Photography Review

### Capturing 3-D Work

```
ESSENTIAL TECHNIQUES:

LIGHTING:
- Directional (creates shadow)
- Reveals form and texture
- Not flat or harsh
- Consistent across views

ANGLES:
- Primary: Most informative
- Secondary: Complementary info
- Consider all viewpoints
- Choose strongest

BACKGROUND:
- Clean, neutral
- Sufficient contrast
- No distractions
- Professional presentation

SCALE:
- Include when needed
- Subtle reference
- Consistent method
- Aids understanding
```

### Common 3-D Photo Errors

```
FIX BEFORE SUBMITTING:

FLAT LIGHTING:
- Add directional light
- Create form-revealing shadows
- Show dimensionality

MISSING ANGLES:
- Multiple views essential
- Don't assume one is enough
- Show full experience

POOR BACKGROUNDS:
- Use seamless paper
- Remove clutter
- Neutral colors
- Professional appearance

SCALE UNCLEAR:
- Add subtle reference
- Context may help
- Consistent approach
- Essential for some work

FORM DOESN'T READ:
- Better lighting
- Different angle
- More contrast
- Re-photograph entirely
```
