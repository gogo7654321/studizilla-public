## ÔøΩüìã Unit Overview

The Protestant Reformation shattered the religious unity of Western Christianity that had existed for a thousand years. What began as Martin Luther's theological dispute became a revolutionary movement that transformed European politics, society, and culture. The Catholic Church responded with its own reform movement, and Europe descended into decades of devastating religious warfare! ‚õ™

## ‚úùÔ∏è Martin Luther and the Reformation

### Martin Luther (1483-1546)

| Aspect | Details |
|--------|---------|
| **Background** | Augustinian monk; professor at Wittenberg |
| **Crisis** | Anxiety about salvation |
| **Discovery** | Justification by faith alone |

| **Target** | Indulgence sales (Johann Tetzel) |

### The Indulgence Controversy

| Aspect | Details |
|--------|---------|
| **Indulgences** | Church documents reducing purgatory time |
| **Tetzel's sales** | "When the coin in the coffer rings, the soul from purgatory springs" |
| **Purpose** | Fund St. Peter's Basilica |
| **Luther's objection** | Not scriptural; exploited faithful |

### Luther's Key Doctrines

| Doctrine | Meaning |
|----------|---------|
| **Sola fide** | Salvation by faith alone |
| **Sola scriptura** | Scripture as sole authority |
| **Sola gratia** | Salvation by grace alone |
| **Priesthood of all believers** | No special priestly class |
| **Two kingdoms** | Secular and spiritual distinct |

### Spread of Luther's Ideas

| Factor | Description |
|--------|-------------|
| **Printing** | Pamphlets spread rapidly |
| **Vernacular** | German rather than Latin |
| **Anti-Rome sentiment** | Political appeal |
| **Urban support** | Cities adopted Lutheranism |
| **Princely protection** | Frederick the Wise |

### Key Events

| Event | Date | Significance |
|-------|------|--------------|
| **95 Theses** | 1517 | Began Reformation |
| **Leipzig Debate** | 1519 | Luther challenged papal authority |
| **Papal bull (Exsurge Domine)** | 1520 | Threatened excommunication |
| **Diet of Worms** | 1521 | Luther refused to recant; declared outlaw |
| **Wartburg Castle** | 1521-22 | Luther hid; translated Bible |

### Diet of Worms (1521)

| Aspect | Details |
|--------|---------|
| **Presiding** | Emperor Charles V |
| **Luther's response** | "Here I stand, I can do no other" |
| **Edict of Worms** | Declared Luther an outlaw/heretic |
| **Protection** | Frederick of Saxony hid him |

### Lutheran Beliefs and Practices

| Aspect | Lutheran | Catholic |
|--------|----------|----------|

| **Sacraments** | 2 (Baptism, Eucharist) | 7 |

| Aspect | Details |
|--------|---------|

### St. Bartholomew's Day Massacre (1572)

| Aspect | Details |
|--------|---------|
| **Date** | August 24, 1572 |
| **Context** | Protestant leaders in Paris for royal wedding |
| **Deaths** | 3,000 in Paris; 10,000+ nationwide |
| **Ordered by** | Catherine de' Medici; Charles IX |
| **Significance** | Radicalized both sides |

### Edict of Nantes (1598)

| Term | Description |
|------|-------------|
| **Issued by** | Henry IV |
| **Huguenot rights** | Limited freedom of worship |
| **Fortified cities** | Huguenots could keep some |
| **Civil equality** | Same rights as Catholics |
| **Significance** | First major religious toleration |
| **Revoked** | 1685 by Louis XIV |

| Aspect | Details |
|--------|---------|
| **Event** | Protestant nobles threw Catholic officials out window |
| **Survived** | Fell into manure pile |
| **Significance** | Started Bohemian revolt |

### Key Figures

| Figure | Role |
|--------|------|

| **Albrecht von Wallenstein** | Imperial general |
| **Cardinal Richelieu** | French minister; supported Protestants |

### Peace of Westphalia (1648)

| Term | Description |
|------|-------------|
| **Calvinism recognized** | Added to Augsburg settlement |
| **Territorial sovereignty** | Princes control religion |
| **Swiss independence** | Recognized |
| **Dutch independence** | Recognized |
| **France gains** | Alsace |
| **Sweden gains** | Baltic territories |
| **Holy Roman Empire** | Weakened; 300+ states |

### Significance of Westphalia

| Impact | Description |
|--------|-------------|
| **End of religious wars** | Religion no longer main cause |
| **State sovereignty** | Modern international system |
| **Balance of power** | New diplomatic concept |
| **Decline of Habsburg power** | Spain and Empire weakened |
| **Rise of France** | Dominant power |

### Thirty Years' War: Consequences

| Consequence | Description |
|-------------|-------------|
| **Population loss** | Germany lost 15-30% of population |
| **Economic devastation** | Agriculture; trade destroyed |
| **Political fragmentation** | Germany remained divided |
| **End of religious wars** | Secularization of politics |
| **Military revolution** | New tactics; standing armies |
