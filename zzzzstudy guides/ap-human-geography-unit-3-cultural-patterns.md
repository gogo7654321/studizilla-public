## ÔøΩüìã Unit Overview

Culture shapes how people interact with their environment and each other. This unit explores the geography of culture - how cultural traits originate, spread, and create distinctive landscapes. From language and religion to food and music, understanding cultural geography helps explain the diversity and conflict in our world! üåç

## üé≠ What is Culture?

### Defining Culture

| Aspect | Description |
|--------|-------------|
| **Definition** | Shared beliefs, values, practices of a group |
| **Learned** | Passed through generations |
| **Dynamic** | Changes over time |
| **Spatial** | Varies by location |

### Components of Culture

| Component | Examples |
|-----------|----------|
| **Material culture** | Buildings; tools; clothing |
| **Non-material culture** | Ideas; beliefs; languages |

|-------|-------------|---------|

## üì° Cultural Diffusion

### Types of Diffusion

| Type | Description | Example |
|------|-------------|---------|
| **Relocation** | Migration spreads culture | Immigrants bring food |
| **Expansion** | Spreads without migration | Ideas spread |

### Expansion Diffusion Types

| Type | Description | Example |
|------|-------------|---------|
| **Hierarchical** | Spreads through hierarchy | Fashion from Paris |
| **Contagious** | Spreads to nearby areas | Disease; slang |
| **Stimulus** | Idea spreads; modified | McDonald's menu variations |

### Reverse Hierarchical Diffusion

| Concept | Description |
|---------|-------------|
| **Definition** | Spreads from lower to higher levels |
| **Example** | Hip-hop from streets to mainstream |
| **Process** | Grassroots to elite adoption |

### Barriers to Diffusion

| Barrier | Description |
|---------|-------------|
| **Physical** | Mountains; oceans |
| **Cultural** | Language; religion |
| **Political** | Borders; policies |
| **Economic** | Poverty; access |
| **Perceptual** | Beliefs; resistance |

|---------|------------|
| **Acculturation** | Adopting some traits from another culture |
| **Assimilation** | Complete absorption into dominant culture |
| **Syncretism** | Blending of cultures into new forms |
| **Multiculturalism** | Coexistence of multiple cultures |

### Cultural Outcomes

| Outcome | Description |
|---------|-------------|
| **Creolization** | New hybrid culture |
| **Cultural convergence** | Cultures becoming similar |
| **Cultural divergence** | Cultures becoming different |
| **Nativism** | Resistance to foreign cultures |

## üì∫ Folk vs. Popular Culture

### Folk Culture

| Characteristic | Description |
|----------------|-------------|
| **Scale** | Local; regional |
| **Change** | Slow |
| **Transmission** | Family; community |
| **Origin** | Unknown; traditional |
| **Examples** | Folk music; traditional dress |

### Popular Culture

| Characteristic | Description |
|----------------|-------------|
| **Scale** | Mass; global |
| **Change** | Rapid |
| **Transmission** | Media; commercial |
| **Origin** | Known; MDCs |
| **Examples** | Pop music; jeans |

### Folk Culture Threats

| Threat | Description |
|--------|-------------|
| **Globalization** | Dominant cultures spread |
| **Media** | Television; internet |
| **Urbanization** | Traditional ways lost |
| **Economics** | Commercial culture profitable |

### Popular Culture Issues

| Issue | Description |
|-------|-------------|
| **Homogenization** | Cultures becoming similar |

| **Loss of diversity** | Local cultures replaced |
