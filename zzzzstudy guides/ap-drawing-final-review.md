## üì∏ Photography Tips

### Capturing Drawings

```
PHOTOGRAPHY ESSENTIALS:

LIGHTING:
- Diffused, even light
- No direct flash
- Minimize shadows
- Reduce glare

FOR GRAPHITE/CHARCOAL:
- Angle light carefully
- Avoid shine
- Polarizing filter helps
- Even illumination

FOR COLOR MEDIA:
- Accurate white balance
- True color reproduction
- Consistent lighting
- No color casts

DETAILS:
- Show mark quality
- Capture texture
- Reveal technique
- Close enough for marks
```

### Common Errors

```
FIX BEFORE SUBMITTING:

GLARE:
- Reposition lights
- Use diffused light
- Shoot at slight angle
- Polarizing filter

UNEVEN EXPOSURE:
- Even lighting
- No bright spots
- Consistent across work
- Check histogram

VALUE INACCURACY:
- Calibrate monitor
- Proper exposure
- Full range visible
- Blacks are black

POOR DETAILS:
- Get close enough
- Sharp focus
- Show actual marks
- Reveal technique
```
