## üß† Line of Reasoning

### Definition

| Line of reasoning | The logical progression of claims that builds an argument |
|-------------------|----------------------------------------------------------|

### Components

| Component | Description |
|-----------|-------------|
| **Thesis** | Central claim |
| **Topic sentences** | Claim of each paragraph |
| **Evidence** | Support for claims |
| **Commentary** | Explanation of connections |

### Strong Line of Reasoning

| Characteristic | Description |
|----------------|-------------|
| **Logical** | Each point follows naturally |
| **Cohesive** | Ideas connect clearly |
| **Progressive** | Builds toward conclusion |
| **Unified** | All parts support thesis |

| Problem | Description |

## üì¢ Claims

### Types of Claims

| Type | Definition | Example |
|------|------------|---------|
| **Main claim (thesis)** | Central argument | "Social media harms teen mental health" |
| **Supporting claims** | Sub-arguments | "Instagram increases anxiety" |
| **Counterclaims** | Opposing views | "Some say social media connects people" |

### Characteristics of Strong Claims

| Characteristic | Description | Example |
|----------------|-------------|---------|
| **Arguable** | Can be debated | "We should ban smartphones in schools" |
| **Specific** | Clear and focused | "Teens should limit social media to 1 hour daily" |
| **Defensible** | Can be supported | "School uniforms reduce bullying" |
| **Significant** | Matters to audience | "Climate change threatens coastal cities" |

### Claims vs. Facts

| Claims | Facts |
|--------|-------|
| "Homework is harmful" | "Students receive 2 hours of homework" |
| "Democracy is best" | "The US is a democracy" |
| "Art class should be required" | "Many schools offer art classes" |

## üìä Evidence

### Types of Evidence

| Type | Description | Example |
|------|-------------|---------|
| **Statistics** | Numerical data | "80% of teens use social media" |
| **Examples** | Specific instances | "In 2020, California banned..." |
| **Expert testimony** | Authority opinions | "Dr. Smith argues that..." |
| **Anecdotes** | Personal stories | "My friend experienced..." |
| **Facts** | Verifiable information | "The law passed in 1965" |
| **Analogies** | Comparisons | "Like a house needs a foundation..." |

### Evaluating Evidence

| Quality | Strong | Weak |
|---------|--------|------|
| **Relevance** | Directly supports claim | Tangentially related |
| **Credibility** | Reliable source | Questionable source |
| **Sufficiency** | Adequate amount | Insufficient support |
| **Currency** | Recent | Outdated |

### Using Evidence Effectively

| Strategy | Description |
|----------|-------------|
| **Integrate smoothly** | Embed in sentences |
| **Cite properly** | Give credit |
| **Vary types** | Mix statistics, examples, etc. |
| **Explain significance** | Connect to claim |

### Evidence Integration

| Method | Example |
|--------|---------|
| **Quote** | Smith states, "Education is key" (45) |
| **Paraphrase** | Smith argues education matters (45) |
| **Summary** | Smith's chapter discusses education (45-60) |

## üí¨ Commentary

### Definition

| Commentary | Explanation of how evidence supports claims |
|------------|---------------------------------------------|

### Purpose of Commentary

| Purpose | Description |
|---------|-------------|
| **Connect** | Link evidence to claim |
| **Explain** | Clarify significance |
| **Analyze** | Interpret meaning |
| **Persuade** | Strengthen argument |

### Commentary vs. Evidence

| Evidence | Commentary |
|----------|------------|
| "80% of students prefer..." | "This data reveals..." |
| "The law states..." | "This provision demonstrates..." |

### Strong Commentary

| Characteristic | Example |
|----------------|---------|

### Weak Commentary

| Problem | Example |

## üìù Thesis Statements

### Definition

| Thesis | The central claim of an essay |
|--------|-------------------------------|

### Characteristics of Strong Thesis

| Characteristic | Description |
|----------------|-------------|
| **Arguable** | Can be debated |
| **Specific** | Clear and focused |
| **Complex** | Acknowledges nuance |
| **Defensible** | Can be supported |
| **Clear** | Easy to understand |

### Thesis Placement

| Position | When to Use |
|----------|-------------|
| **End of introduction** | Most common |
| **Beginning** | Direct approach |
| **Multiple sentences** | Complex arguments |

### Types of Thesis

| Type | Description | Example |
|------|-------------|---------|
| **Simple** | Single claim | "Homework should be banned" |
| **Complex** | Claim + reasons | "Homework should be banned because it increases stress, reduces family time, and doesn't improve learning" |
| **Qualified** | Claim + conditions | "While homework can reinforce skills, it should be limited to 30 minutes to prevent stress" |

### Thesis Formula for Argument

| Formula | Example |
|---------|---------|
| **Claim + Because** | "Schools should start later because teens need more sleep" |
| **Although X, Y** | "Although sports teach discipline, they shouldn't be prioritized over academics" |
| **X; therefore, Y** | "Teen mental health is declining; therefore, we must regulate social media" |

### Thesis Formula for Rhetorical Analysis

| Formula | Example |
|---------|---------|
| **Speaker + Purpose + Strategy** | "In her speech, Obama uses personal anecdotes and emotional appeals to inspire young voters" |

|------|---------|---------|
| **Introduction** | Engage, provide context, state thesis | Hook + background + thesis |
| **Body paragraphs** | Develop claims | Topic sentence + evidence + commentary |

| **Logical** | Cause-effect, problem-solution | Explanatory arguments |
| **Compare/contrast** | Similarities and differences | Comparative analysis |

## üìä The Toulmin Model of Argument

### Core Components

| Component | Definition | Example |
|-----------|------------|---------|
| **Claim** | Main assertion | "Schools should start later" |
| **Data/Evidence** | Support for claim | "Studies show teens need 9 hours sleep" |
| **Warrant** | Logical connection | "More sleep improves performance" |
| **Backing** | Support for warrant | "Research on adolescent brain development" |
| **Qualifier** | Limits to claim | "In most cases..." |
| **Rebuttal** | Anticipated objection | "Unless extracurricular scheduling prevents it" |

## üß† Logical Fallacies to Avoid

### Fallacies of Relevance

| Fallacy | Definition | Example |
|---------|------------|---------|
| **Ad hominem** | Attacking the person | "Don't listen to him‚Äîhe's ugly" |
| **Appeal to authority** | Unqualified expert | "This actor endorses this medicine" |
| **Appeal to emotion** | Only using feelings | "Think of the children!" |
| **Bandwagon** | Everyone does it | "Everyone has one, so I need one" |
| **Red herring** | Distracting topic | Changing subject when losing |

### Fallacies of Insufficient Evidence

| Fallacy | Definition | Example |
|---------|------------|---------|
| **Hasty generalization** | Too few examples | "I met two rude New Yorkers, so all are rude" |
| **False cause** | Correlation ‚â† causation | "Crime rose after video games became popular" |
| **Slippery slope** | Assuming chain reaction | "If we ban plastic bags, all freedom ends" |
| **Straw man** | Misrepresenting opponent | "My opponent wants open borders" (when they don't) |
| **False dilemma** | Only two options | "You're either with us or against us" |

### Fallacies of Clarity

| Fallacy | Definition | Example |
|---------|------------|---------|
| **Equivocation** | Shifting word meaning | "The law says men are equal, so women aren't" |
| **Circular reasoning** | Conclusion = premise | "The Bible is true because it says so" |
| **Loaded question** | Assumes conclusion | "When did you stop cheating?" |

## üèõÔ∏è Classical Argument Structure

### Six Parts of Classical Argument

| Part | Latin | Function | Example |
|------|-------|----------|---------|
| **Introduction** | Exordium | Capture attention | Hook + context |
| **Background** | Narratio | Provide context | History of issue |
| **Proposition** | Propositio | State thesis | Your main claim |
| **Proof** | Confirmatio | Present evidence | Body paragraphs |
| **Refutation** | Refutatio | Address opponents | Counterarguments |
| **Conclusion** | Peroratio | Call to action | Final appeal |

### Modern Adaptations

| Format | Structure | Best For |
|--------|-----------|----------|
| **Classical** | Intro ‚Üí Background ‚Üí Thesis ‚Üí Proof ‚Üí Refutation ‚Üí Conclusion | Formal arguments |
| **Rogerian** | Intro ‚Üí Common ground ‚Üí Opposition ‚Üí Your position ‚Üí Compromise | Controversial topics |
| **Inverted pyramid** | Thesis first ‚Üí Supporting details ‚Üí Background | Journalism |
| **Problem-solution** | Problem ‚Üí Causes ‚Üí Solution ‚Üí Benefits | Policy arguments |

## ‚úçÔ∏è Sample Argument Outlines

### Classical Outline Example

```
I. Exordium (Introduction)
   - Hook: Startling statistic about student sleep
   - Context: Current school start times
   - Thesis: High schools should start no earlier than 8:30 AM

II. Narratio (Background)
   - History of school schedules
   - Current research on teen sleep

III. Confirmatio (Proof)
   A. Claim 1: Academic performance improves
      - Evidence: Stanford study data
      - Commentary: Explains correlation
   B. Claim 2: Mental health benefits
      - Evidence: CDC statistics
      - Commentary: Significance
   C. Claim 3: Reduced accident rates
      - Evidence: State comparisons
      - Commentary: Implications

IV. Refutatio (Refutation)
   - Counterargument: Sports scheduling
   - Response: Schools have adapted successfully

V. Peroratio (Conclusion)
   - Restate thesis
   - Call to action
   - Broader significance
```
