## ‚ùì Research Question Review

### Question Quality

```
STRONG RESEARCH QUESTIONS:

ORIGINAL:
Not already answered
Adds to knowledge

FOCUSED:
Specific and manageable
Clear scope

COMPLEX:
Not yes/no
Requires investigation

RESEARCHABLE:
Data can be gathered
Sources exist

SIGNIFICANT:
Matters to field
Has implications
```

### Question Types

```
QUESTION FRAMEWORKS:

CAUSAL:
"What effect does X have on Y?"

COMPARATIVE:
"How does X compare to Y?"

CORRELATIONAL:
"What is the relationship between X and Y?"

EVALUATIVE:
"How effective is X?"

EXPLORATORY:
"What factors contribute to X?"

## üó£Ô∏è Oral Defense Review

### Preparing for Defense

```
DEFENSE PREPARATION:

KNOW EVERYTHING:
- Every source
- Method choices
- Results meaning
- Limitations
- Implications

ANTICIPATE QUESTIONS:
- Why this method?
- Alternative interpretations?
- Limitations?
- Future research?
- Applications?

PRACTICE:
- Have someone quiz you
- Time responses
- Stay calm
- Think before speaking
```

### Common Questions

```
TYPICAL DEFENSE QUESTIONS:

METHODOLOGY:
"Why did you choose this approach?"
"How would different methods affect results?"

INTERPRETATION:
"What alternative explanations exist?"
"How might others interpret this?"

LIMITATIONS:
"What are the limitations?"
"How would you address them?"

EXTENSION:
"How might this apply to...?"
"What would you research next?"

REFLECTION:
"What would you do differently?"
"What did you learn from the process?"
```

### Response Strategies

```
ANSWERING QUESTIONS:

STRUCTURE:
1. Acknowledge question
2. Pause to think
3. Answer directly
4. Provide support
5. Extend if appropriate

PHRASES:
"That's a great question..."
"Based on my research..."
"The data suggest..."
"One limitation is..."
"For future research..."

HANDLING DIFFICULTY:
"That's beyond my scope, but..."
"I would hypothesize..."
"I'd need to investigate further..."
```

### Defense Scoring

```
ORAL DEFENSE (within 25%):

HIGH SCORE:
- Thoroughly addresses questions
- Defends with evidence
- Extends thoughtfully
- Shows deep understanding
- Acknowledges limitations

LOW SCORE:
- Limited response
- Minimal support
- Surface understanding
- Cannot extend
- Avoids questions
