```
Complete self-contained piece
```

## ğŸ”„ 8.2 Binary Form

### Simple Binary (AB)

```
BINARY FORM:
Two main sections (A and B)

SIMPLE BINARY:
A    ||:   B   :||
No repeat of A at end

CHARACTERISTICS:
- A usually ends on V (or in dominant key)
- B returns to I (tonic)
- B often longer than A
- Related thematic material
```

### Rounded Binary (ABA')

```
ROUNDED BINARY:
A returns at end of B section

||: A :||: B â†’ A' :||

A: Opening material (ends V or :V)
B: Contrasting/development
A': Return of A (often abbreviated)
    Ends in tonic

COMMON IN:
- Baroque dance movements
- Classical minuet and trio
- Theme in variations
```

### Binary Form Analysis

```
BINARY ANALYSIS STEPS:

1. FIND THE DOUBLE BAR
   Major structural division

2. CHECK A ENDING
   HC or in dominant = continuous
   PAC in tonic = sectional

3. LOOK FOR A RETURN
   A material in second half?
   Yes = Rounded
   No = Simple

4. CHECK PROPORTIONS
   A and B length
   Note repetitions
```

## ğŸ­ 8.3 Ternary Form

### Simple Ternary (ABA)

```
TERNARY FORM:
Three-part structure with return

A    B    A

A: First section (complete in itself)
B: Contrasting section (different key, mood)
A: Return of first section (same or varied)

DIFFERENCE FROM ROUNDED BINARY:
- Ternary: B is independent section
- Rounded binary: A' is part of B section
- Ternary: Clear sectional divisions
```

### Compound Ternary

```
COMPOUND TERNARY:
Each section is itself a form

MINUET AND TRIO:
Minuet (A) | Trio (B) | Minuet da capo (A)

Minuet = binary (or rounded binary)
Trio = binary (or rounded binary)
Minuet da capo = Minuet without repeats

STRUCTURE:
||: a :||: b â†’ a' :|| ||: c :||: d â†’ c' :|| ||: a :||: b â†’ a' :||
    Minuet              Trio              Minuet D.C.
```

### Ternary Analysis

```
TERNARY VS. BINARY:

TERNARY:
- B stands alone
- Clear section breaks
- A returns complete
- Usually: A ends PAC, B ends PAC, A ends PAC

BINARY (ROUNDED):
- A' is conclusion of B
- Continuous flow
- A' often abbreviated
- Usually: A ends HC, Bâ†’A' ends PAC
```

## ğŸ” 8.4 Rondo Form

### Five-Part Rondo (ABACA)

```
FIVE-PART RONDO:

A    B    A    C    A

A: Refrain (returns multiple times)
B: First episode (contrasting)
A: Refrain return
C: Second episode (different contrast)
A: Final refrain

CHARACTERISTICS:
- A always in tonic
- B and C in related keys
- Light, playful character often
```

### Seven-Part Rondo (ABACABA)

```
SEVEN-PART RONDO:

A    B    A    C    A    B    A

More elaborate form
B returns before final A
Common in Classical finales

VARIATION:
Sonata-Rondo (hybrid form)
Combines rondo with sonata elements
```

### Rondo Analysis

```
RONDO IDENTIFICATION:

1. FIND THE REFRAIN
   Recurring A section
   Always returns to tonic
   
2. COUNT EPISODES
   How many different B, C sections?
   
3. MAP THE FORM
   A-B-A = not rondo (ternary)
   A-B-A-C-A = rondo
   
4. CHECK KEY SCHEME
   Refrain = tonic
   Episodes = related keys
```

## ğŸ¼ 8.5 Sonata Form

### Exposition

```
SONATA FORM EXPOSITION:

||: EXPOSITION :||

FIRST THEME (Primary):
- Tonic key
- Main thematic material
- May be period or sentence

TRANSITION:
- Modulates to new key
- Often unstable
- Leads to second theme

SECOND THEME (Secondary):
- New key (V or III)
- Contrasting character
- Often more lyrical

CLOSING THEME:
- Confirms new key
- Cadential material
- Leads to repeat or development
```

### Development

```
DEVELOPMENT SECTION:

FREE/UNSTABLE SECTION:
- Manipulates exposition themes
- Multiple key areas
- Sequences, fragmentation

TECHNIQUES:
- Thematic fragmentation
- Sequential passages
- New key explorations
- Contrapuntal textures

RETRANSITION:
- Prepares tonic return
- Often dominant pedal
- Builds tension
- Leads to recapitulation
```

### Recapitulation

```
RECAPITULATION:

"RECAP" = Return + Resolution

FIRST THEME:
- Returns in tonic
- Often identical or similar

TRANSITION:
- Modified to STAY in tonic
- Key adjustment

SECOND THEME:
- NOW IN TONIC (major change!)
- Resolves key tension

CLOSING/CODA:
- Confirms tonic
- May add new cadential material

CODA (optional):
- Additional closing section
- After main form complete
```

## ğŸ­ 8.6 Theme and Variations

### Variation Form

```
THEME AND VARIATIONS:

Theme | Var. 1 | Var. 2 | Var. 3 | ... | Final Var.

THEME:
Usually simple, memorable tune
Often binary or rounded binary
Clearly stated first

VARIATIONS:
Each presents theme differently
Maintain essential elements
Increase complexity often
```

### Variation Techniques

```
TYPES OF VARIATION:

MELODIC:
- Ornamentation
- New figurations
- Rhythmic changes

HARMONIC:
- New harmonization
- Mode change (majorâ†”minor)
- Chromatic alterations

TEXTURAL:
- Change of register
- Different voicing
- Counterpoint added

RHYTHMIC:
- New rhythmic patterns
- Tempo changes
- Meter changes

CHARACTER:
- Style changes (march, waltz)
- Dramatic contrasts
```

### Analyzing Variations

```
VARIATION ANALYSIS:

FOR EACH VARIATION, IDENTIFY:

1. WHAT STAYS SAME
   - Harmonic structure?
   - Melodic outline?
   - Phrase structure?

2. WHAT CHANGES
   - Texture?
   - Rhythm?
   - Character?

3. PROGRESSIVE ELEMENT
   - Does complexity increase?
   - Is there a climax?
   
4. FINALE
   - Return to theme?
   - Coda or grand finale?
```

## ğŸ“Š 8.7 Other Forms

### Strophic Form

```
STROPHIC:
Same music, different text (verses)

A    A    A    A

Common in:
- Folk songs
- Hymns
- Pop verses (without chorus)

MODIFIED STROPHIC:
A    A'   A''  A'''
Slight variations each verse
```

### Through-Composed

```
THROUGH-COMPOSED:
No large-scale repetition

A    B    C    D    ...

Continuous new material
Text-driven (often art song)
Each section different

Examples:
- Schubert's "ErlkÃ¶nig"
- Opera scenes
- Some art songs
```

### Verse-Chorus Form

```
VERSE-CHORUS (Pop form):

Intro | Verse | Chorus | Verse | Chorus | Bridge | Chorus | Outro

VERSE: Different lyrics, similar music
CHORUS: Same lyrics and music (hook)
BRIDGE: Contrasting section
PRE-CHORUS: Optional transition

Common in:
- Pop music
- Rock music
- Musical theater
```
