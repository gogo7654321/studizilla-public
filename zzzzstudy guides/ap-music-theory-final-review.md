## üéì Key Concepts by Topic

### Pitch Organization

```
1. SCALES PROVIDE FRAMEWORK
   - Major and minor systems

   - Tonic is home
   - Dominant creates tension
   - Subdominant moves away

   - Melodic intervals = horizontal
   - Harmonic intervals = vertical
   - Quality determines color
```

### Harmonic Language

```
1. TRIADS ARE FOUNDATION
   - Four qualities
   - Three inversions
   - Diatonic functions

2. SEVENTH CHORDS ADD COLOR
   - V7 most important
   - Five main types
   - Resolve tendency tones

3. PROGRESSIONS FOLLOW LOGIC
   - T ‚Üí S ‚Üí D ‚Üí T
```

### Melodic Elements

```
1. CONTOUR CREATES SHAPE
   - Balance steps and leaps
   - Compensate after leaps
   - Build to climax

2. NCTs EMBELLISH
   - Don't define harmony
   - Add motion and color
   - Resolve predictably

3. PHRASES STRUCTURE MUSIC
   - Period = question + answer
   - Cadences mark endings
   - Sentences develop ideas
```

### Chromatic Harmony

```
1. SECONDARY DOMINANTS TONICIZE
   - V/X creates temporary tonic
   - Raised notes = leading tones
   - Resolve like V ‚Üí I

2. MODULATION CHANGES KEY

   - Closely related easiest

3. SPECIAL CHORDS ADD DRAMA
   - Neapolitan = ‚ô≠II6
   - Augmented 6ths expand to V
```

### Form

```

   - A sections return

   - Keys relate

2. CONTRAST CREATES INTEREST
   - B sections differ
```
