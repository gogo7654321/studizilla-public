## üèôÔ∏è Urban Models

### Concentric Zone Model (Burgess)

| Zone | Description |
|------|-------------|
| **1 - CBD** | Central Business District |
| **2 - Transition** | Industry; deteriorating housing |
| **3 - Working class** | Older housing; near work |
| **4 - Middle class** | Newer housing; residential |
| **5 - Commuter** | Suburbs; higher income |

### Burgess Model Features

| Aspect | Description |
|--------|-------------|
| **Created** | 1925 |
| **Based on** | Chicago |
| **Key idea** | Zones radiate from CBD |
| **Assumptions** | Single center; uniform land; competition |
| **Criticism** | Oversimplified; dated |

### Sector Model (Hoyt)

| Concept | Description |
|---------|-------------|
| **Created** | 1939 |
| **Key idea** | Land use extends in wedges/sectors |
| **Factor** | Transportation routes |
| **Pattern** | Similar uses cluster along routes |

### Multiple Nuclei Model (Harris & Ullman)

| Concept | Description |
|---------|-------------|
| **Created** | 1945 |
| **Key idea** | Multiple centers of activity |
| **Nuclei** | Different specialized areas |
| **More realistic** | For modern cities |

### Multiple Nuclei Features

| Center Type | Example |
|-------------|---------|
| **CBD** | Downtown |
| **Industrial** | Factory district |
| **Retail** | Shopping centers |
| **Suburban** | Edge cities |
| **Airport** | Aviation hub |

### Galactic City/Urban Realms Model

| Concept | Description |
|---------|-------------|
| **Created** | 1960s-1990s |
| **Key idea** | Decentralized metropolitan areas |
| **Features** | Edge cities; suburbs dominant |
| **Reality** | Many American metros today |

## üè¢ Urban Land Use

### Central Business District (CBD)

| Feature | Description |
|---------|-------------|
| **Function** | Commercial heart of city |
| **Land values** | Highest in metro |
| **Buildings** | Tall; intensive use |
| **Access** | Transportation hub |
| **Challenges** | Competition from suburbs |

### Bid-Rent Theory

| Concept | Description |
|---------|-------------|
| **Idea** | Land value decreases from center |
| **Reason** | Accessibility premium |
| **Commercial** | Pay most for CBD |
| **Industrial** | Mid-range locations |
| **Residential** | Outward locations |

### Land Use Zones

| Zone | Characteristics |
|------|-----------------|
| **Commercial** | Retail; offices; services |
| **Industrial** | Manufacturing; warehouses |
| **Residential** | Housing (varying density) |
| **Mixed-use** | Combination of uses |
| **Open space** | Parks; recreation |

### Zoning

| Concept | Description |
|---------|-------------|
| **Definition** | Legal land use regulations |
| **Purpose** | Separate incompatible uses |
| **Types** | Residential, commercial, industrial |
| **Criticism** | Can promote segregation; sprawl |

## üå≥ Urban Sprawl

### Characteristics

| Feature | Description |
|---------|-------------|
| **Low density** | Spread-out development |
| **Car dependence** | Must drive everywhere |
| **Segregated uses** | Separated zones |
| **Leap-frog development** | Discontinuous |

### Causes of Sprawl

| Cause | Description |
|-------|-------------|
| **Cheap land** | At periphery |
| **Highway construction** | Enables commuting |
| **Zoning** | Separates uses |
| **Housing preferences** | Larger homes |
| **Tax policies** | Favor new construction |

### Costs of Sprawl

| Cost | Description |
|------|-------------|
| **Environmental** | Habitat loss; pollution |
| **Infrastructure** | Expensive to extend services |
| **Traffic** | Congestion; commute time |
| **Health** | Sedentary; obesity |

| Principle | Description |
|-----------|-------------|

| **Compact development** | Higher density |
| **Walkability** | Pedestrian-friendly |
| **Transit-oriented** | Near public transit |

| Concept | Description |
|---------|-------------|
| **Definition** | Design movement for walkable communities |

| Concept | Description |
|---------|-------------|
| **Definition** | Development around transit stations |
| **Features** | High density; mixed use; walkable |
| **Benefits** | Reduce car dependence |
| **Examples** | Portland; Arlington, VA |

| Concept | Description |
|---------|-------------|
