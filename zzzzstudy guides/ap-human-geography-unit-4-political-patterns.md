## ÔøΩüìã Unit Overview

Political geography examines how humans organize space and govern territory. From the concept of the nation-state to the drawing of boundaries and the role of international organizations, understanding political geography is essential for making sense of conflict, cooperation, and governance in our world! üèõÔ∏è

### Essential Questions

| Question | Focus |
|----------|-------|
| What is a state? | Sovereignty; territory |
| How are states shaped? | Boundaries; morphology |
| How do states organize internally? | Federalism; devolution |
| What role do international organizations play? | Cooperation; governance |
| Why do territorial conflicts occur? | Boundaries; resources; identity |

### Key Concepts

| Concept | Description |
|---------|-------------|
| **State** | Sovereign political unit |
| **Nation** | Cultural group |
| **Boundary** | Dividing line between states |
| **Sovereignty** | Supreme authority within territory |

## üåê States and Nations

### Key Definitions

| Term | Definition |
|------|------------|
| **State** | Political unit with defined territory, government, sovereignty |
| **Nation** | Group sharing cultural characteristics |
| **Nation-state** | State where boundaries match nation |
| **Country** | Common term for state |
| **Sovereignty** | Complete control over territory |

### Nation-State Concept

| Aspect | Description |
|--------|-------------|
| **Ideal** | One nation = one state |
| **Reality** | Very few true nation-states |
| **Examples** | Japan; Iceland (closest to ideal) |
| **Most states** | Multinational |

### Types of States

| Type | Definition | Examples |
|------|------------|----------|
| **Nation-state** | One nation = one state | Japan, Iceland |
| **Multinational state** | Multiple nations in one state | Russia, India |
| **Multistate nation** | One nation in multiple states | Kurds, Koreans |
| **Stateless nation** | Nation without a state | Kurds, Palestinians |

### Examples of Stateless Nations

| Nation | Location |
|--------|----------|
| **Kurds** | Turkey, Iraq, Syria, Iran |
| **Palestinians** | Israel, occupied territories |
| **Tibetans** | China |
| **Basques** | Spain, France |
| **Rohingya** | Myanmar (stateless people) |

| Era | Development |
|-----|-------------|

### Colonialism and States

| Era | Pattern |
|-----|---------|
| **Pre-colonial** | Traditional boundaries |
| **Colonial** | European powers drew boundaries |

## üìè Boundaries

### Types of Boundaries

| Type | Definition | Example |
|------|------------|---------|
| **Natural/Physical** | Follows natural features | Mountains, rivers |
| **Geometric** | Straight lines; coordinates | US-Canada 49th parallel |
| **Cultural** | Follows cultural divisions | Language, religion |

### Physical Boundaries

| Feature | Examples |
|---------|----------|
| **Mountains** | Andes (Chile-Argentina); Pyrenees (Spain-France) |
| **Rivers** | Rio Grande (US-Mexico); Rhine |
| **Lakes** | Great Lakes (US-Canada) |
| **Deserts** | Sahara |
| **Issues** | Rivers change course; mountains don't isolate |

### Geometric Boundaries

| Type | Examples |
|------|----------|
| **Latitude/Longitude** | 49th parallel; Korean DMZ (38th parallel) |
| **Straight lines** | Many African borders |
| **Arcs** | Some state borders |
| **Issues** | Often ignore cultural realities |

### Antecedent vs. Subsequent Boundaries

| Type | Definition | Example |
|------|------------|---------|
| **Antecedent** | Drawn before settlement | US-Canada |
| **Subsequent** | Drawn after settlement | India-Pakistan |
| **Superimposed** | Forced on existing cultures | Colonial Africa |
| **Relic** | No longer functions | Berlin Wall location |

### Boundary Disputes

| Type | Definition | Example |
|------|------------|---------|
| **Positional** | Location of line | India-China |
| **Territorial** | Who owns area | Kashmir; Crimea |
| **Resource** | Control of resources | Oil fields; water |
| **Functional** | How boundary operates | Immigration policy |

### Famous Boundary Disputes

| Dispute | Parties | Issue |
|---------|---------|-------|
| **Kashmir** | India, Pakistan, China | Territory |
| **South China Sea** | China, neighbors | Islands; resources |

## üî∑ State Morphology

### Shape Types

| Shape | Definition | Advantages | Disadvantages | Example |
|-------|------------|------------|---------------|---------|
| **Compact** | Roughly circular | Easy to govern | - | Poland |
| **Elongated** | Long and narrow | Access to resources | Hard to govern | Chile |
| **Prorupted** | Compact with extension | Access to feature | Vulnerable extension | Thailand |
| **Fragmented** | Separate pieces | - | Hard to unify | Indonesia |
| **Perforated** | Surrounds another | - | Depends on neighbor | South Africa/Lesotho |

### Enclaves and Exclaves

| Term | Definition | Example |
|------|------------|---------|
| **Enclave** | Surrounded by another state | Vatican City; Lesotho |
| **Exclave** | Separated from main territory | Alaska; Kaliningrad |

### Landlocked States

| Aspect | Description |
|--------|-------------|
| **Definition** | No coastline |
| **Challenges** | Trade; access to ocean |
| **Examples** | Bolivia, Mongolia, Ethiopia |
| **Consequences** | Economic disadvantage |

## ‚öñÔ∏è Internal Organization

### Unitary vs. Federal States

| Type | Definition | Examples |
|------|------------|----------|
| **Unitary** | Central government holds power | France, Japan, China |
| **Federal** | Power divided between central and regional | USA, Germany, India |

### Federalism

| Aspect | Description |
|--------|-------------|
| **Definition** | Power divided between levels |
| **Advantages** | Local control; diversity |
| **Disadvantages** | Complexity; inequality |
| **Examples** | US, Canada, Australia, Germany |

### Unitary States

| Aspect | Description |
|--------|-------------|
| **Definition** | Central government controls all |
| **Advantages** | Uniform policies; efficiency |
| **Disadvantages** | May ignore local needs |
| **Examples** | France, UK, Japan |

### Devolution

| Aspect | Description |
|--------|-------------|
| **Definition** | Transfer of power to local levels |
| **Causes** | Cultural differences; regional identity |
| **Examples** | Scotland, Catalonia, Quebec |
| **Results** | Autonomy; sometimes independence movements |

### Causes of Devolution

| Cause | Description | Example |
|-------|-------------|---------|
| **Ethnic** | Cultural differences | Scotland, Catalonia |
| **Economic** | Regional wealth/poverty | Northern Italy |
| **Spatial** | Physical isolation | Hawaii from mainland |
| **Historical** | Past independence | Baltics from USSR |

## üåç Centripetal vs. Centrifugal Forces

### Centripetal Forces (Unifying)

| Force | Description | Example |
|-------|-------------|---------|
| **Nationalism** | Shared identity | US patriotism |
| **Common language** | Communication | French in France |
| **Religion** | Shared beliefs | Islam in Saudi Arabia |
| **Shared history** | Common narrative | Revolutionary history |
| **External threat** | Common enemy | Unity during war |
| **National symbols** | Flags, anthems | |
| **Sports** | National teams | World Cup |

### Centrifugal Forces (Dividing)

| Force | Description | Example |
|-------|-------------|---------|
| **Ethnic differences** | Multiple groups | Bosnia |
| **Language divisions** | Different tongues | Belgium |
| **Religious divisions** | Different faiths | India-Pakistan |
| **Economic inequality** | Rich vs. poor regions | Northern vs. Southern Italy |
| **Peripheral location** | Distance from core | Catalonia |

### Irredentism

| Aspect | Description |
|--------|-------------|
| **Definition** | Claiming territory in another state |
| **Based on** | Ethnic/historical claims |
| **Examples** | Russia-Crimea; Serbia-Kosovo |
| **Consequences** | Conflict; territorial disputes |

### Secessionism

| Aspect | Description |
|--------|-------------|
| **Definition** | Movement to break away |
| **Examples** | Scotland, Catalonia, Quebec |
| **Success** | South Sudan, Kosovo (disputed) |
| **Failure** | Biafra, Chechnya |

## üèõÔ∏è Electoral Geography

### Redistricting

| Concept | Description |
|---------|-------------|
| **Definition** | Redrawing electoral districts |
| **Purpose** | Reflect population changes |
| **Frequency** | After census (every 10 years in US) |

### Gerrymandering

| Concept | Description |
|---------|-------------|
| **Definition** | Drawing districts for political advantage |
| **Types** | Packing; cracking; hijacking; kidnapping |
| **Consequences** | Unrepresentative elections |
| **Issue** | Democracy concerns |

### Gerrymandering Techniques

| Technique | Description |
|-----------|-------------|
| **Packing** | Concentrate opponents in few districts |
| **Cracking** | Spread opponents across many districts |
| **Hijacking** | Force two incumbents to compete |
| **Kidnapping** | Move incumbent out of district |

### Reapportionment

| Concept | Description |
|---------|-------------|
| **Definition** | Reallocating seats based on population |
| **US example** | House seats among states |
| **Frequency** | After each census |

## ‚öñÔ∏è Federal vs. Unitary States Comparison

### Detailed Comparison

| Aspect | Federal States | Unitary States |
|--------|---------------|----------------|
| **Power distribution** | Divided between levels | Centralized |
| **Constitution** | Defines power division | May delegate temporarily |
| **Regional governments** | Have constitutional powers | Powers granted by center |
| **Legal systems** | May vary by region | Uniform |
| **Taxation** | Multiple levels | Primarily central |
| **Education policy** | Often regional | National standards |
| **Examples** | USA, Germany, India, Brazil | France, Japan, China, UK |

### Advantages and Disadvantages

| Type | Advantages | Disadvantages |
|------|------------|---------------|
| **Federal** | Local autonomy; diversity; experimentation | Complexity; inequality; coordination |
| **Unitary** | Efficiency; equality; coordination | May ignore local needs; less democratic |

## üåç Geopolitical Theories

### Classical Geopolitics

| Theory | Theorist | Year | Core Idea |
|--------|----------|------|-----------|
| **Heartland Theory** | Halford Mackinder | 1904 | Control of Eurasian heartland = world power |
| **Rimland Theory** | Nicholas Spykman | 1942 | Control of Eurasian rim more important |
| **Sea Power** | Alfred Mahan | 1890 | Naval dominance determines power |
| **Organic State** | Friedrich Ratzel | 1897 | States as living organisms; lebensraum |

### Modern Geopolitical Concepts

| Concept | Description | Examples |
|---------|-------------|----------|
| **Buffer state** | Between rival powers | Mongolia (China-Russia), Nepal (China-India) |
| **Shatterbelt** | Zone of persistent conflict | Middle East, Balkans |
| **Chokepoint** | Strategic narrow passage | Strait of Hormuz, Malacca, Suez |
| **Sphere of influence** | Area of dominant external power | Russian near abroad, Monroe Doctrine |

### Strategic Chokepoints

| Chokepoint | Location | Strategic Importance |
|------------|----------|---------------------|
| **Strait of Hormuz** | Persian Gulf | 21% of world oil |
| **Strait of Malacca** | Singapore | 25% of global trade |
| **Suez Canal** | Egypt | Europe-Asia shortcut |
| **Panama Canal** | Panama | Atlantic-Pacific link |
| **Bab el-Mandeb** | Yemen-Djibouti | Red Sea entrance |
| **Turkish Straits** | Turkey | Black Sea access |
