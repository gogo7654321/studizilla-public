## ‚ùì Why Do We Forget?

### Theories of Forgetting

| Theory | Explanation |
|--------|-------------|
| **Encoding failure** | Never properly encoded |
| **Storage decay** | Memory trace fades |
| **Retrieval failure** | Can't access stored memory |
| **Interference** | Other memories block retrieval |

### Interference Types

| Type | Definition | Example |
|------|------------|---------|
| **Proactive** | Old interferes with new | Old phone # blocks new |
| **Retroactive** | New interferes with old | New password blocks old |

### Memory Aid

- **Pro**active = **P**revious blocks new
- **Retro**active = **R**ecent blocks old

| Type | Definition | Example |
|------|------------|---------|

| Concept | Explanation |
|---------|-------------|
| Prototypes | Most typical members |
| Comparison | Compare new items to prototype |

| Typical (Prototype) | Atypical |
|--------------------|----------|

| Strategy | Definition | Pros/Cons |

| Heuristic | Definition | Example |
|-----------|------------|---------|

|------------|------------------------------|

## ‚öñÔ∏è Decision Making

### Availability Heuristic

| Works When | Fails When |
|------------|-----------|
| Frequent events come to mind | Memorable events aren't frequent |
| Example | Overestimate plane crashes |

### Representativeness Heuristic

| Works When | Fails When |
|------------|-----------|
| Stereotypes are accurate | Base rates matter |
| Example | Ignore that there are more farmers than librarians |

Harry studied French in high school but now struggles to remember it after years of studying Spanish. This is an example of:

People overestimate the likelihood of dramatic events like plane crashes because:

## ‚úçÔ∏è Sample Free Response

**Question**: Sara is studying for an important exam. Using your knowledge of memory, explain how each of the following could affect her performance:

(a) Spacing effect
(b) Context-dependent memory
(c) Semantic encoding
(d) Proactive interference
(e) Testing effect

**Answer**:

(a) **Spacing effect**: If Sara spreads her studying over several days rather than cramming the night before, she will have better retention. The spacing effect shows that distributed practice leads to stronger, more durable memories than massed practice. This allows time for consolidation and creates more retrieval opportunities.

(b) **Context-dependent memory**: If Sara studies in an environment similar to where she will take the exam (same room, similar lighting, same desk), she may recall information better. Context-dependent memory suggests that retrieval is enhanced when the physical context during encoding matches the context during retrieval.

(c) **Semantic encoding**: If Sara focuses on understanding the meaning of the material rather than just memorizing facts superficially, she will remember more. Deep processing (semantic level) creates stronger memory traces than shallow processing (just reading words). Connecting new information to existing knowledge improves retention.

(d) **Proactive interference**: If Sara has previously learned similar but different material, that old learning may interfere with her ability to recall the new information. For example, if she learned different formulas in a previous course, those old formulas might intrude when she tries to remember the new ones.

(e) **Testing effect**: If Sara quizzes herself while studying rather than just rereading her notes, she will remember more on the actual exam. The act of retrieving information strengthens memory more than passive review. Practice tests also help identify gaps in knowledge.
