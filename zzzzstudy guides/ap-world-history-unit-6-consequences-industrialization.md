## ÔøΩüìã Unit Overview

Industrialization transformed not only European economies but the entire global order. The search for raw materials, markets, and national prestige drove a new wave of imperialism that brought much of Africa and Asia under European control. This period also witnessed massive migrations, new ideologies, and the beginnings of resistance movements that would reshape the 20th century! üè≠

### Essential Questions

| Question | Focus |
|----------|-------|
| How did industrialization spread? | Technology transfer, adaptation |
| What drove New Imperialism? | Economic, political, ideological motives |
| How did empires control colonies? | Direct rule, indirect rule, settler colonies |
| How did people respond to imperialism? | Resistance, adaptation, reform |
| How did migration patterns change? | Push/pull factors, destinations |

### Key Themes

| Theme | Examples |
|-------|----------|
| **Industrialization spreads** | Europe, US, Japan |
| **New Imperialism** | Scramble for Africa, Asia |
| **Nationalism** | Unification, ethnic movements |
| **Migration** | Global labor movements |
| **Resistance** | Reform movements, rebellions |

## üè≠ Spread of Industrialization

### Why Britain Led

| Factor | Advantage |
|--------|-----------|
| **Resources** | Coal, iron abundant |
| **Capital** | Investment available |
| **Labor** | Enclosure freed workers |
| **Empire** | Markets, raw materials |
| **Institutions** | Banks, stable government |
| **Geography** | Island; rivers; harbors |

### Second Industrial Revolution (1870-1914)

| Innovation | Impact |
|------------|--------|
| **Steel** | Bessemer process; stronger, cheaper |
| **Electricity** | Power, lighting, machinery |
| **Chemicals** | Fertilizers, dyes, explosives |
| **Petroleum** | Internal combustion engine |
| **Communications** | Telegraph, telephone |
| **Transportation** | Railroads, steamships |

### Industrialization by Country

| Country | Period | Key Industries | State Role |
|---------|--------|----------------|------------|
| **Britain** | 1760s+ | Textiles, iron, steam | Minimal |
| **Belgium** | 1820s | Textiles, coal | Some support |
| **France** | 1830s | Luxury, later heavy | Moderate |
| **Germany** | 1840s+ | Steel, chemicals, electrical | Strong |
| **United States** | 1840s+ | Railroads, steel, oil | Land grants, tariffs |
| **Russia** | 1890s | Railroads, heavy industry | State-led |
| **Japan** | 1870s+ | Textiles, later heavy | Meiji government led |

### Japanese Industrialization

| Aspect | Details |
|--------|---------|
| **Context** | Perry's arrival (1853) |
| **Response** | Meiji Restoration (1868) |
| **Slogan** | "Rich country, strong army" |
| **Government role** | Built, then sold factories |
| **Borrowed** | Western technology, experts |
| **Zaibatsu** | Industrial conglomerates |
| **Result** | Only non-Western industrializer |

### Russian Industrialization

| Aspect | Details |
|--------|---------|
| **Timing** | 1890s under Witte |
| **State role** | Heavy involvement |
| **Focus** | Railroads (Trans-Siberian) |
| **Finance** | Foreign investment |
| **Problems** | Serfdom (until 1861); limited market |
| **Result** | Partial, uneven development |

## üåê New Imperialism (1870-1914)

### What Was Different?

| Old Imperialism (1500-1800) | New Imperialism (1870-1914) |
|-----------------------------|----------------------------|
| Trade focused | Territorial control |
| Coastal presence | Interior penetration |
| Limited technology edge | Overwhelming military advantage |
| Gradual expansion | Rapid conquest |
| Few powers | Many European competitors |

### Motivations for Imperialism

#### Economic Motives

| Motive | Explanation |
|--------|-------------|
| **Raw materials** | Cotton, rubber, copper, oil |
| **Markets** | Sell manufactured goods |
| **Investment** | Profitable opportunities |
| **Strategic resources** | Coal stations, naval bases |

#### Political Motives

| Motive | Explanation |
|--------|-------------|
| **National prestige** | Empire = power |
| **Competition** | Other powers expanding |
| **Strategic** | Protect trade routes |
| **Nationalism** | Glory, unity at home |

#### Ideological Motives

| Motive | Explanation |
|--------|-------------|
| **"Civilizing mission"** | Spread Western culture |
| **Social Darwinism** | "Superior" races dominate |
| **Christianity** | Missionary impulse |
| **Racism** | Justified exploitation |

### Technology Enabling Conquest

| Technology | Use |
|------------|-----|
| **Steamships** | Navigate rivers into interiors |
| **Quinine** | Prevent malaria |
| **Machine guns** | Maxim gun; overwhelming firepower |
| **Telegraph** | Rapid communication |
| **Railroads** | Move troops, extract resources |
| **Breech-loading rifles** | Superior to muskets |

## üåç Imperialism in Africa

### Before the Scramble

| Aspect | Details |
|--------|---------|
| **European presence** | Coastal trading posts |
| **African kingdoms** | Many strong states |
| **Slave trade ending** | Abolition 1807-1888 |
| **Interior** | Largely unknown to Europeans |
| **Exploration** | Livingstone, Stanley |

### Berlin Conference (1884-1885)

| Aspect | Details |
|--------|---------|
| **Called by** | Otto von Bismarck |
| **Purpose** | Prevent European war over Africa |
| **No Africans** | Not invited |
| **Rules** | "Effective occupation" required |
| **Result** | Africa carved up |
| **Outcome** | By 1914, only Ethiopia, Liberia independent |

### Colonial Powers in Africa

| Power | Territories |
|-------|-------------|
| **Britain** | Egypt, Sudan, Kenya, Uganda, Nigeria, South Africa, Rhodesia |
| **France** | West Africa, North Africa, Madagascar |
| **Belgium** | Congo (Leopold's personal colony) |
| **Germany** | Tanzania, Namibia, Cameroon, Togo |
| **Portugal** | Angola, Mozambique |
| **Italy** | Libya, Eritrea, Somalia |
| **Spain** | Morocco (part), Western Sahara |

### Belgian Congo

| Aspect | Details |
|--------|---------|
| **Owner** | King Leopold II (personal) |
| **Resource** | Rubber, ivory |
| **Methods** | Forced labor, mutilation, terror |
| **Death toll** | Millions died |
| **Exposure** | E.D. Morel, Roger Casement |
| **Transfer** | To Belgian government (1908) |

### Effects on Africa

| Effect | Description |
|--------|-------------|
| **Boundaries** | Arbitrary; divided ethnic groups |
| **Economy** | Export-oriented; cash crops |
| **Labor** | Forced; migrant; contract |
| **Culture** | Christianity; European education |
| **Politics** | Traditional leaders subordinated |
| **Infrastructure** | Railroads, ports (for extraction) |
| **Resistance** | Uprisings suppressed |

### African Resistance

| Resistance | Location | Outcome |
|------------|----------|---------|
| **Zulu Wars** | South Africa | Defeated after victories |
| **Ashanti Wars** | Gold Coast | Eventually conquered |
| **Maji Maji Rebellion** | German East Africa | Crushed; thousands died |
| **Ethiopia** | Horn of Africa | Defeated Italy at Adwa (1896) |
| **Samori Ture** | West Africa | Resisted French for decades |

### Ethiopia's Success

| Factor | Explanation |
|--------|-------------|
| **Leadership** | Emperor Menelik II |
| **Modernization** | Acquired European weapons |
| **Unity** | Mobilized diverse groups |
| **Battle of Adwa (1896)** | Decisively defeated Italy |
| **Result** | Remained independent |
| **Significance** | Symbol for African resistance |

## üåè Imperialism in Asia

### British India

| Period | Development |
|--------|-------------|
| **1757** | Plassey; EIC control begins |
| **1857** | Sepoy Mutiny |
| **1858** | Crown rule (British Raj) |
| **1876** | Victoria "Empress of India" |
| **1885** | Indian National Congress |
| **1947** | Independence |

### East India Company to Raj

| Phase | Characteristics |
|-------|-----------------|
| **EIC rule** | Company controlled territory |
| **Sepoy Mutiny** | Indian soldiers rebelled |
| **British Raj** | Direct crown control |
| **Administration** | Indian Civil Service |
| **Economy** | Deindustrialization; raw exports |

### Sepoy Mutiny/Rebellion (1857)

| Cause | Description |
|-------|-------------|
| **Cartridges** | Greased with beef/pork fat |
| **Grievances** | Religion, pay, treatment |
| **Annexations** | Doctrine of Lapse |
| **Spread** | Northern India |
| **Suppression** | Brutal British response |
| **Result** | End of EIC; crown rule |

### Effects of British Rule in India

| Effect | Description |
|--------|-------------|
| **Economic** | Deindustrialization; textile imports |
| **Infrastructure** | Railroads, telegraph, ports |
| **Education** | English; Western system |
| **Administration** | Civil service; law codes |
| **Famines** | Export policy worsened |
| **Political** | Nationalism emerged |

### China and Imperialism

| Event | Date | Significance |
|-------|------|--------------|
| **Opium War I** | 1839-1842 | Treaty of Nanjing |
| **Taiping Rebellion** | 1850-1864 | 20-30 million died |
| **Opium War II** | 1856-1860 | More concessions |
| **Self-Strengthening** | 1861-1895 | Modernization attempt |
| **Sino-Japanese War** | 1894-1895 | Japan victory |
| **Boxer Rebellion** | 1900 | Anti-foreign crushed |

### Opium Wars

| Aspect | Details |
|--------|---------|
| **British problem** | Trade deficit with China |
| **Solution** | Sell Indian opium to China |
| **Chinese response** | Banned opium; seized stocks |
| **British response** | War (1839) |
| **Result** | Treaty of Nanjing (1842) |

### Treaty of Nanjing (1842)

| Term | Significance |
|------|--------------|
| **Hong Kong** | Ceded to Britain |
| **Treaty ports** | 5 cities opened |
| **Extraterritoriality** | Foreigners under own law |
| **Indemnity** | China paid costs |
| **Tariffs** | Fixed low |
| **"Unequal treaty"** | Model for others |

### Spheres of Influence in China

| Power | Sphere |
|-------|--------|
| **Britain** | Yangtze Valley, Hong Kong |
| **France** | South China |
| **Germany** | Shandong |
| **Russia** | Manchuria |
| **Japan** | Fujian, later more |
| **US** | Open Door Policy (no sphere) |

### Chinese Responses

| Response | Description |
|----------|-------------|
| **Taiping Rebellion** | Christian-influenced; nearly toppled Qing |
| **Self-Strengthening** | "Chinese learning, Western technology" |
| **Hundred Days Reform** | 1898; crushed by conservatives |
| **Boxer Rebellion** | Anti-foreign; crushed by powers |
| **Revolution** | 1911; end of imperial China |

### Japan: Different Path

| **Russo-Japanese War (1905)** | Defeated Russia |

### Southeast Asia

## üö¢ Global Migration

### Types of Migration

| Type | Description | Examples |
|------|-------------|----------|
| **Settler migration** | Europeans to colonies | Americas, Australia, South Africa |
| **Labor migration** | Contract workers | Indians to Caribbean, Chinese to US |
| **Forced migration** | Slave trade (declining) | Atlantic, Indian Ocean |
| **Internal migration** | Rural to urban | Within industrializing countries |
| **Refugees** | Fleeing persecution | Irish famine, Russian pogroms |

### Push and Pull Factors

| Push (Leave) | Pull (Attract) |
|--------------|----------------|
| Poverty | Economic opportunity |
| Famine | Land availability |
| Persecution | Religious freedom |
| Overpopulation | Labor demand |
| War, instability | Political stability |
| Enclosure | Free/cheap land |

### European Emigration

| Period | Numbers |
|--------|---------|
| **1820-1860** | ~5 million |
| **1860-1914** | ~30 million |
| **Total 1815-1914** | ~60 million |

| Origin | Destinations |
|--------|--------------|
| **Britain** | US, Canada, Australia |
| **Ireland** | US (especially after famine) |
| **Germany** | US (Midwest) |
| **Italy** | US, Argentina, Brazil |
| **Eastern Europe** | US (late 1800s+) |
| **Scandinavia** | US (upper Midwest) |

### Asian Migration

| Group | Destinations | Work |
|-------|--------------|------|
| **Chinese** | US (railroad), SE Asia, Americas | Mining, railroad, labor |
| **Indians** | Caribbean, Africa, SE Asia | Plantation indenture |
| **Japanese** | Hawaii, US, Brazil | Agriculture |

### Indentured Servitude

| Aspect | Details |
|--------|---------|
| **What** | Contract labor for passage |
| **Who** | Indians, Chinese mainly |
| **Where** | Caribbean, Africa, SE Asia, Pacific |
| **Terms** | 3-5 years; passage paid |
| **Conditions** | Often harsh; debt bondage |
| **Numbers** | Millions in 1800s |
| **After slavery** | Replaced slave labor |

### Effects of Migration

| Effect | Description |
|--------|-------------|
| **Americas** | Population boom; diversity |
| **Sending countries** | Remittances; brain drain |
| **Culture** | Diaspora communities |
| **Nativism** | Anti-immigrant reactions |
| **Labor** | Cheap, flexible workforce |
