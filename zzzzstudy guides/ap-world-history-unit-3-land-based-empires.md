## ÔøΩüìã Unit Overview

The period from 1450-1750 witnessed the rise of powerful land-based empires that used gunpowder technology, centralized bureaucracies, and religious legitimacy to control vast territories. These empires transformed political, economic, and cultural landscapes across Eurasia! üí•

### Essential Questions

| Question | Focus |
|----------|-------|
| How did gunpowder change warfare? | Military technology and tactics |
| How did empires legitimize rule? | Religion, bureaucracy, culture |
| How did empires manage diversity? | Religious tolerance, administration |
| What caused imperial expansion? | Military, economic, political factors |
| How did empires decline? | Internal and external pressures |

### Major Empires of this Period

| Empire | Region | Religion | Peak Period |
|--------|--------|----------|-------------|
| **Ottoman** | Middle East, SE Europe, N. Africa | Sunni Islam | 1520-1566 |

## üïå The Ottoman Empire (1299-1922)

### Origins and Rise

The Ottoman Empire began as a small Turkish state in Anatolia and grew to become one of the longest-lasting and most powerful empires in world history.

| Period | Development | Significance |
|--------|-------------|--------------|
| **1299** | Founded by Osman I | Small Turkish principality |
| **1326** | Captured Bursa | First major capital |
| **1361** | Captured Adrianople (Edirne) | European foothold |
| **1389** | Battle of Kosovo | Balkans conquered |
| **1453** | Fall of Constantinople | Byzantine Empire ended |
| **1517** | Conquered Egypt | Claimed caliphate |
| **1520-1566** | Suleiman the Magnificent | Empire at peak |

### The Conquest of Constantinople (1453)

| Aspect | Details |
|--------|---------|
| **Sultan** | Mehmed II "The Conqueror" |
| **Technology** | Massive cannons breached walls |
| **Significance** | End of Byzantine Empire |
| **Renamed** | Istanbul |
| **Hagia Sophia** | Converted to mosque |
| **Impact** | Ottoman control of trade routes |

### Government and Administration

The Ottomans developed a sophisticated administrative system that allowed them to rule diverse populations effectively.

| Institution | Function | Details |
|-------------|----------|---------|
| **Sultan** | Absolute ruler | Combined secular and religious authority |
| **Grand Vizier** | Chief minister | Ran day-to-day government |
| **Divan** | Imperial council | Advisory body to sultan |
| **Pashas** | Provincial governors | Administered provinces |
| **Kadis** | Islamic judges | Applied sharia law |
| **Ulama** | Religious scholars | Legitimized rule |

### The Devshirme System

One of the most distinctive features of Ottoman administration was the devshirme, or "collection" system.

| Feature | Description |
|---------|-------------|
| **What** | Levy of Christian boys from Balkans |
| **Age** | Typically 8-18 years old |
| **Conversion** | Boys converted to Islam |
| **Education** | Trained in palace schools |
| **Outcomes** | Janissaries or administrators |
| **Purpose** | Create loyal servant class |
| **Effect** | Merit-based advancement |

### The Janissaries

| Aspect | Details |
|--------|---------|
| **Origin** | Devshirme system |
| **Training** | Elite military education |
| **Loyalty** | Directly to sultan |
| **Weapons** | First to use firearms extensively |
| **Privileges** | Salaries, exemptions |
| **Later problems** | Became political force |
| **Abolished** | 1826 by Mahmud II |

### The Millet System

The Ottomans managed religious diversity through the millet system.

| Concept | Description |
|---------|-------------|
| **Definition** | Self-governing religious communities |
| **Millets** | Greek Orthodox, Armenian, Jewish |
| **Self-governance** | Own courts for personal law |
| **Leadership** | Religious leaders represented community |
| **Taxes** | Jizya (poll tax) for non-Muslims |
| **Limitations** | Some dress codes, restrictions |
| **Effect** | Relative religious tolerance |

### Suleiman the Magnificent (1520-1566)

| Achievement | Details |
|-------------|---------|
| **Military** | Conquered Hungary, besieged Vienna |
| **Legal** | "Lawgiver" - codified laws |
| **Architecture** | Commissioned Suleymaniye Mosque |
| **Culture** | Patronized arts, poetry |
| **Administration** | Peak of Ottoman efficiency |
| **Territory** | Largest extent of empire |

### Ottoman Economy

| Sector | Features |
|--------|----------|
| **Agriculture** | Tax farming (iltizam) |
| **Trade** | Controlled East-West routes |
| **Manufacturing** | Textiles, ceramics |
| **Finance** | Sophisticated tax system |
| **Guilds** | Regulated urban crafts |

### Ottoman Military Technology
