## ‚ùì 1.1 Developing Inquiry Questions

### What Makes a Good Question

```
STRONG INQUIRY QUESTIONS:

COMPLEX:
Not answered with simple yes/no
Requires investigation and analysis

RESEARCHABLE:
Can find evidence to explore
Sources exist to consult

DEBATABLE:
Multiple perspectives possible
Reasonable people disagree

SIGNIFICANT:
Matters to someone
Has real-world implications

FOCUSED:
Specific enough to explore deeply
Not too broad or too narrow
```

### Question Types

```
QUESTION PROGRESSION:

FACTUAL QUESTIONS:
- What is...?
- When did...?
- Who was...?
‚Üí Answered with facts (starting point)

ANALYTICAL QUESTIONS:
- Why did...?
- How does...?
- What causes...?
‚Üí Requires analysis (going deeper)

EVALUATIVE QUESTIONS:
- Should...?
- To what extent...?
- What is the best...?
‚Üí Requires judgment (most complex)

MOVE FROM FACTUAL ‚Üí ANALYTICAL ‚Üí EVALUATIVE
```

### Refining Questions

```
QUESTION REFINEMENT PROCESS:

1. START BROAD
   "Climate change"
   
2. ADD SPECIFICITY
   "Effects of climate change"
   
3. NARROW FOCUS
   "Effects of climate change on agriculture"
   
4. ADD COMPLEXITY
   "How should farmers adapt to climate 
    change effects on crop yields?"
   
5. CONSIDER PERSPECTIVES
   "How do proposed climate adaptations 
    for agriculture balance economic 
    viability with environmental 
    sustainability?"
```

### Question Quality Checklist

```
EVALUATE YOUR QUESTION:

‚ñ° Is it open-ended (not yes/no)?
‚ñ° Does it require multiple sources?
‚ñ° Are there multiple perspectives?
‚ñ° Is it appropriately scoped?
‚ñ° Does it matter (significance)?
‚ñ° Can you research it?
‚ñ° Does it spark curiosity?
‚ñ° Is it original (not already settled)?
```

### Research Strategies

```
FINDING SOURCES:

ACADEMIC DATABASES:
- JSTOR
- Academic Search Complete
- Google Scholar
- ProQuest
- Specialized databases

LIBRARY RESOURCES:
- Catalog searches
- Reference librarian help
- Interlibrary loan
- Special collections

CITATION MINING:
- Check source bibliographies
- Find who cites key articles
- Follow the conversation
- Build your source network

SEARCH TECHNIQUES:
- Boolean operators (AND, OR, NOT)
- Quotation marks for phrases
- Truncation (*)
- Filters for date, type
```

### Evaluating Sources: SIFT Method

```
SIFT EVALUATION:

S - STOP
   Before using, stop and think
   What do you know about this source?

I - INVESTIGATE THE SOURCE
   Who created this?
   What is their expertise?
   What is their motivation?

F - FIND BETTER COVERAGE
   Can you find this elsewhere?
   What do other sources say?
   Is there a better version?

T - TRACE CLAIMS
   Where does the info originate?
   Can you verify the claims?
   What's the evidence chain?
```

## üîç 1.3 Evaluating Evidence

### Evidence Quality Criteria

```
EVALUATING EVIDENCE:

RELEVANCE:
- Does it address your question?
- Is it directly connected?
- Does it support your point?

RELIABILITY:
- Is the source trustworthy?
- Are methods sound?
- Is information verified?

SUFFICIENCY:
- Is there enough evidence?
- Are there multiple sources?
- Is the sample adequate?

CURRENCY:
- Is it recent enough?
- Is older data still valid?
- Is there newer information?

BIAS AWARENESS:
- What perspective is represented?
- What might be missing?
- How might bias affect accuracy?
```

### Types of Evidence

```
EVIDENCE TYPES:

STATISTICAL DATA:
- Numbers and statistics
- Quantitative research
- Polls and surveys
- Measurable information

EXPERT TESTIMONY:
- Expert opinions
- Professional analysis
- Authoritative statements
- Specialized knowledge

EXAMPLES/CASES:
- Specific instances
- Case studies
- Illustrative examples
- Real-world applications

TEXTUAL EVIDENCE:
- Quotations
- Paraphrased ideas
- Referenced arguments
- Documentary evidence

VISUAL EVIDENCE:
- Images, charts, graphs
- Diagrams
- Maps
- Infographics
```

### Bias Recognition

```
IDENTIFYING BIAS:

SELECTION BIAS:
What's included/excluded?
Why these examples?

CONFIRMATION BIAS:
Does source only show supporting evidence?
Are counterarguments addressed?

LANGUAGE BIAS:
What words are used?
Loaded language?
Emotional appeals?

STRUCTURAL BIAS:
Who produced this?
What are their interests?
Who funds them?

IMPLICIT BIAS:
Unstated assumptions?
Cultural assumptions?
Unconscious perspectives?
```

## üìù 1.4 Note-Taking and Organization

### Effective Note-Taking

```
NOTE-TAKING STRATEGIES:

FOR EACH SOURCE, RECORD:
1. Full citation information
2. Key arguments/claims
3. Relevant evidence
4. Useful quotations (with page numbers)
5. Your reactions/questions
6. Connections to other sources

ANNOTATION APPROACH:
- Summarize main points
- Note your reactions
- Ask questions
- Make connections
- Evaluate credibility
```

### Source Organization

```
ORGANIZING RESEARCH:

CITATION MANAGEMENT:
- Use citation software
  (Zotero, EasyBib, etc.)
- Track all sources
- Note where you found them
- Include access dates

THEMATIC ORGANIZATION:
- Group by topic
- Track perspectives
- Note connections
- Identify gaps

SYNTHESIS MATRIX:
Create a table:
| Theme    | Source 1 | Source 2 | Source 3 |
|----------|----------|----------|----------|
| Topic A  | Says X   | Says Y   | Says Z   |
| Topic B  | ...      | ...      | ...      |
```

### Avoiding Plagiarism

```
ACADEMIC INTEGRITY:

MUST CITE:
- Direct quotations
- Paraphrased ideas
- Statistics and data
- Unique concepts
- Images and visuals

HOW TO AVOID PLAGIARISM:
1. Keep careful notes on sources
2. Quote exactly (with citation)
3. Paraphrase in your own words
4. Synthesize multiple sources
5. When in doubt, cite

PROPER PARAPHRASE:
- Use your own words
- Change sentence structure
- Still cite the source
- Don't just swap synonyms
```

## ü§î 1.5 Critical Reading

### Active Reading Strategies

```
READING CRITICALLY:

BEFORE READING:
- Preview the text
- Check author/source
- Identify purpose
- Set your purpose

DURING READING:
- Annotate actively
- Question claims
- Note evidence
- Track argument structure

AFTER READING:
- Summarize main points
- Evaluate arguments
- Connect to other sources
- Identify questions
```

### Identifying Arguments

```
ARGUMENT COMPONENTS:

CLAIM:
What is the author arguing?
What position do they take?

EVIDENCE:
What support is provided?
What types of evidence?

REASONING:
How does evidence connect to claim?
What's the logical structure?

ASSUMPTIONS:
What's taken for granted?
What's not stated but implied?

COUNTERARGUMENTS:
Are opposing views addressed?
How are they handled?
```

### Rhetorical Analysis

```
ANALYZING RHETORIC:

ETHOS (Credibility):
- Author's expertise
- Trustworthiness
- Professional reputation
- Ethical appeal

PATHOS (Emotion):
- Emotional appeals
- Language choices
- Imagery
- Personal stories

LOGOS (Logic):
- Reasoning quality
- Evidence strength
- Argument structure
- Logical connections

KAIROS (Timing):
- Timeliness
- Cultural context
- Relevance to moment
- Opportune argument
```
