```

2. Identify grammatical structures
3. Look for specific vocabulary
```

### Translation Tips

```
Caesar:
1. Find main verb (often at end)
2. Identify ablative absolutes first
3. Handle indirect statement carefully
4. Watch for historic present

2. Untangle poetic word order
3. Find main verb
4. Group adjectives with nouns
5. Watch for elision
```
