# üèõÔ∏è Unit 1: Foundations of American Democracy

## üìã Unit Overview

The foundations of American democracy rest on revolutionary ideas about liberty, limited government, and popular sovereignty. This unit explores the philosophical origins, constitutional design, and key principles that established the American system of government. Understanding these foundations is essential for grasping how American government operates today! üá∫üá∏

| Question | Focus |
|----------|-------|

| Why was the Constitution created? | Articles' weaknesses |

| What limits government power? | Separation; checks |
| How does the Constitution change? | Amendments; interpretation |

### Key Concepts

| Concept | Description |
|---------|-------------|
| **Popular sovereignty** | Power from the people |
| **Limited government** | Constitution constrains power |

## ‚öñÔ∏è Articles of Confederation

### Features

| Feature | Description |
|---------|-------------|
| **Adopted** | 1781 |
| **Structure** | Unicameral Congress |
| **States' power** | Strong; sovereignty |
| **Central government** | Weak |
| **Executive** | None |
| **Judiciary** | None |

### Weaknesses

| Weakness | Problem |
|----------|---------|
| **No executive** | Can't enforce laws |
| **No judiciary** | Can't resolve disputes |
| **No tax power** | Can't raise revenue |
| **No commerce regulation** | Trade conflicts |
| **Amendment** | Requires unanimous consent |
| **Legislation** | Requires 9/13 states |

### Shays' Rebellion (1786-1787)

| Aspect | Description |
|--------|-------------|
| **What** | Armed uprising in Massachusetts |
| **Who** | Indebted farmers |
| **Cause** | Economic hardship; foreclosures |
| **Response** | Federal government couldn't respond |
| **Significance** | Showed Articles' weakness |
| **Result** | Push for Constitutional Convention |

## üìú Constitutional Convention (1787)

### Context

| Factor | Description |
|--------|-------------|
| **Purpose** | Originally to revise Articles |
| **Location** | Philadelphia |
| **Attendees** | 55 delegates; 12 states (not Rhode Island) |
| **Key figures** | Madison, Washington, Franklin, Hamilton |
| **Outcome** | New Constitution |

### Major Debates and Compromises

| Issue | Positions | Compromise |
|-------|-----------|------------|
| **Representation** | Virginia Plan vs. New Jersey Plan | Great Compromise |
| **Slavery counting** | South wanted slaves counted | Three-Fifths Compromise |
| **Slave trade** | North vs. South | 20-year protection |
| **Executive** | Single vs. plural | Single president |
| **Selection of president** | Popular vs. legislative | Electoral College |

### Virginia Plan

| Feature | Description |
|---------|-------------|
| **Proposed by** | James Madison |
| **Legislature** | Bicameral |
| **Representation** | Based on population |
| **Favored** | Large states |

### New Jersey Plan

| Feature | Description |
|---------|-------------|
| **Proposed by** | William Paterson |
| **Legislature** | Unicameral |
| **Representation** | Equal per state |
| **Favored** | Small states |

### Great Compromise (Connecticut Compromise)

| Feature | Description |
|---------|-------------|
| **House** | Representation by population |
| **Senate** | Equal representation (2 per state) |
| **Result** | Bicameral Congress |
| **Significance** | Both large and small states satisfied |

### Three-Fifths Compromise

| Issue | Resolution |
|-------|------------|
| **Question** | How to count enslaved people |
| **Formula** | 3/5 of enslaved population |
| **Purpose** | Representation and taxation |
| **Legacy** | Overturned by 14th Amendment |

### Goals of the Preamble

| Goal | Meaning |
|------|---------|
| "Form a more perfect Union" | Improve on Articles |
| "Establish Justice" | Fair legal system |
| "Insure domestic Tranquility" | Internal peace |
| "Provide for the common defence" | National security |
| "Promote the general Welfare" | Public good |
| "Secure the Blessings of Liberty" | Protect freedom |

### Constitutional Articles

|---------|---------|

## üèõÔ∏è Federalism

### Definition

| Concept | Description |
|---------|-------------|
| **Federalism** | Power divided between national and state |
| **Purpose** | Prevent concentration; local control |
| **Dual sovereignty** | Both levels supreme in their spheres |

### Constitutional Basis

| Provision | Effect |
|-----------|--------|
| **Enumerated powers** | Federal powers listed |
| **Reserved powers** | States retain (10th Amendment) |
| **Concurrent powers** | Shared powers |
| **Supremacy Clause** | Federal law supreme |

### Types of Powers

| Type | Description | Examples |
|------|-------------|----------|
| **Enumerated** | Explicitly granted to federal | Coin money; declare war; regulate commerce |
| **Implied** | Necessary and proper | Create bank; draft soldiers |
| **Reserved** | Kept by states | Education; marriage; police power |
| **Concurrent** | Both levels | Tax; borrow; courts |

### Key Clauses

| Clause | Location | Function |
|--------|----------|----------|
| **Necessary and Proper** | Article I, Section 8 | Implied powers |
| **Commerce Clause** | Article I, Section 8 | Regulate interstate commerce |
| **Supremacy Clause** | Article VI | Federal law supreme |
| **10th Amendment** | Bill of Rights | Reserved powers to states |

### Federalism Evolution

| Era | Type | Features |
|-----|------|----------|
| **1789-1932** | Dual federalism | "Layer cake"; separate spheres |
| **1932-1970s** | Cooperative | "Marble cake"; shared; New Deal |
| **1980s+** | New federalism | Return power to states |
| **Current** | Hybrid | Mix of approaches |

### Fiscal Federalism

| Type | Description |
|------|-------------|
| **Categorical grants** | Specific purpose; federal conditions |
| **Block grants** | Broad purpose; state discretion |
| **Mandates** | Federal requirements |
| **Unfunded mandates** | Requirements without funding |

## ‚öîÔ∏è Ratification Debate

### Federalists vs. Anti-Federalists

| Issue | Federalists | Anti-Federalists |
|-------|-------------|------------------|
| **Constitution** | Supported | Opposed |
| **Central government** | Strong needed | Too strong |
| **Bill of Rights** | Unnecessary | Essential |
| **Key figures** | Hamilton, Madison, Jay | Patrick Henry, George Mason |
| **Publications** | Federalist Papers | Various Anti-Federalist writings |

### Federalist Papers

| Number | Author | Topic |
|--------|--------|-------|
| **10** | Madison | Factions; large republic |
| **51** | Madison | Separation of powers; checks |
| **70** | Hamilton | Strong executive |
| **78** | Hamilton | Independent judiciary; judicial review |

### Federalist No. 10 Key Ideas

| Concept | Explanation |
|---------|-------------|
| **Factions** | Groups with common interest adverse to rights of others |
| **Problem** | Factions can threaten liberty |
| **Solution** | Large republic; many factions balance |
| **Extended republic** | Diverse interests prevent tyranny of majority |

### Federalist No. 51 Key Ideas

| Concept | Explanation |
|---------|-------------|
| **Separation of powers** | Each branch separate |
| **Checks and balances** | "Ambition must counteract ambition" |
| **Double security** | Federal and state governments check each other |
| **Republican government** | Protects liberty |

### Brutus No. 1 (Anti-Federalist)

| Concern | Description |
|---------|-------------|
| **Necessary and Proper** | Too broad; unlimited power |
| **Supremacy Clause** | States subordinated |
| **Size** | Large republic unworkable |
| **Representation** | Too few representatives |

### Ratification Process

|-------|--------------|

## üîÑ Amendment Process

### Two Methods of Proposal

| Method | Requirement |
|--------|-------------|
| **Congressional** | 2/3 vote of both houses |
| **Convention** | 2/3 of state legislatures request (never used) |

### Two Methods of Ratification

| Method | Requirement |
|--------|-------------|
| **State legislatures** | 3/4 of states (most common) |
| **State conventions** | 3/4 of states (used once: 21st Amendment) |

### Difficulty of Amendment

| Factor | Significance |
|--------|--------------|
| **Supermajorities** | 2/3 and 3/4 required |
| **Result** | Only 27 amendments in 230+ years |
| **Purpose** | Stability; prevents hasty changes |

## ÔøΩÔ∏è Key Constitutional Clauses Explained

### Commerce Clause (Article I, Section 8)

| Aspect | Description |
|--------|-------------|
| **Text** | "To regulate Commerce... among the several States" |
| **Original purpose** | Prevent trade wars between states |
| **Expansion** | Gibbons v. Ogden (1824); Heart of Atlanta (1964) |
| **Modern use** | Basis for most federal economic regulation |
| **Limits** | United States v. Lopez (1995) |

### Full Faith and Credit Clause (Article IV)

| Aspect | Description |
|--------|-------------|
| **Text** | States must recognize other states' public acts |
| **Purpose** | National unity; prevent forum shopping |
| **Examples** | Marriages; court judgments; contracts |

### Privileges and Immunities Clause (Article IV)

| Aspect | Description |
|--------|-------------|
| **Text** | Citizens of each state entitled to privileges in other states |
| **Purpose** | Prevent discrimination against out-of-staters |
| **Limit** | Reasonable differences allowed |

## ‚öñÔ∏è Foundational Supreme Court Cases

### Marbury v. Madison (1803)

| Aspect | Description |
|--------|-------------|
| **Issue** | William Marbury's commission not delivered |
| **Chief Justice** | John Marshall |
| **Ruling** | Established judicial review |
| **Significance** | Courts can declare laws unconstitutional |
| **Quote** | "It is emphatically the province and duty of the judicial department to say what the law is" |

### McCulloch v. Maryland (1819)

| Aspect | Description |
|--------|-------------|
| **Issue** | Could Congress create a national bank? Could Maryland tax it? |
| **Ruling** | Yes to bank (implied powers); No to tax (supremacy) |
| **Significance** | Broad interpretation of Necessary and Proper Clause |
| **Quote** | "Let the end be legitimate... all means which are appropriate are constitutional" |

### Gibbons v. Ogden (1824)

| Aspect | Description |
|--------|-------------|
| **Issue** | NY steamboat monopoly vs. federal license |
| **Ruling** | Federal commerce power includes navigation |
| **Significance** | Broad interpretation of Commerce Clause |
