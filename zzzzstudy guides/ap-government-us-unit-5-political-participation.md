## ÔøΩüìã Unit Overview

Political participation is the heart of democracy. This unit explores how citizens engage in politics through voting, campaigns, political parties, and interest groups. Understanding these mechanisms is essential for understanding how American democracy functions and how citizens can make their voices heard! üó≥Ô∏è

### Key Concepts

| Concept | Description |

## üó≥Ô∏è Elections and Electoral Systems

### Types of Elections

| Type | Purpose |
|------|---------|
| **Primary** | Nominate party candidates |
| **General** | Choose between parties |
| **Midterm** | Congressional (non-presidential years) |
| **Special** | Fill vacancies |
| **Recall** | Remove official (some states) |

### Primary Types

| Type | Description |
|------|-------------|
| **Closed** | Only registered party members |
| **Open** | Any voter can participate |
| **Semi-closed** | Independents can join |
| **Blanket/Jungle** | All candidates together |
| **Caucus** | Meeting; discussion; vote |

### Electoral College

| Feature | Description |
|---------|-------------|
| **Total electors** | 538 |
| **Needed to win** | 270 |
| **Allocation** | Senators + Representatives |
| **Winner-take-all** | 48 states + DC |
| **Proportional** | Maine and Nebraska |

### How Electoral College Works

| Step | Process |
|------|---------|
| **1** | States hold elections |
| **2** | Winner gets state's electors |
| **3** | Electors meet in states |
| **4** | Votes counted in Congress |
| **5** | Majority wins |

### Electoral College Debates

| Pro | Con |
|-----|-----|
| Federalism | Popular vote loser can win |
| Forces coalition-building | Small states overrepresented |
| State-by-state focus | Swing states dominate |
| Decisive outcomes | Winner-take-all ignores minorities |

### When No Majority

| Scenario | Process |
|----------|---------|
| **President** | House chooses (1 vote per state) |
| **Vice President** | Senate chooses |
| **Precedent** | 1800, 1824 |

## üí∞ Campaign Finance

### Why Money Matters

| Purpose | Use |
|---------|-----|
| **Advertising** | TV, radio, digital |
| **Staff** | Consultants; workers |
| **Travel** | Events; rallies |
| **Research** | Polling; opposition |
| **GOTV** | Get out the vote |

### Campaign Finance Laws

| Law | Year | Provisions |
|-----|------|------------|
| **Federal Election Campaign Act** | 1971, 1974 | Contribution limits; FEC created |
| **Bipartisan Campaign Reform Act (McCain-Feingold)** | 2002 | Banned soft money; regulated issue ads |

### Key Supreme Court Cases

| Case | Ruling |
|------|--------|
| **Buckley v. Valeo (1976)** | Spending is speech; limits on contributions OK |
| **Citizens United v. FEC (2010)** | Corporations/unions have speech rights |
| **McCutcheon v. FEC (2014)** | Struck down aggregate limits |

### Citizens United v. FEC (2010)

| Issue | Ruling |
|-------|--------|
| **Question** | Can government limit corporate political spending? |
| **Ruling** | No; spending is protected speech |
| **Result** | Super PACs enabled |
| **Impact** | Increased money in politics |

### Types of Campaign Money

| Type | Description |
|------|-------------|
| **Hard money** | Regulated; limited contributions |
| **Soft money** | Unregulated party money (banned 2002) |
| **Independent expenditure** | Not coordinated with campaign |
| **Dark money** | Undisclosed donors |

### Political Action Committees (PACs)

| Type | Rules |
|------|-------|
| **Traditional PAC** | $5,000 limit per candidate per election |
| **Super PAC** | Unlimited; no direct coordination |
| **527 organization** | Issue advocacy; less regulated |
| **501(c)(4)** | "Social welfare"; can be anonymous |

### Contribution Limits (2024 cycle)

| Donor | To Candidate | To PAC | To Party |
|-------|--------------|--------|----------|
| **Individual** | $3,300/election | $5,000/year | $41,300/year |
| **PAC** | $5,000/election | $5,000/year | $15,000/year |
| **Super PAC** | $0 (no direct) | N/A | N/A |

## üó∫Ô∏è Gerrymandering Explained

### Types of Gerrymandering

| Type | Description |
|------|-------------|
| **Partisan** | Advantage one party |
| **Racial** | Dilute minority voting power |
| **Incumbent protection** | Keep incumbents safe |

### Gerrymandering Techniques

| Technique | Description |
|-----------|-------------|
| **Cracking** | Split opposition voters among districts |
| **Packing** | Concentrate opposition in few districts |
| **Stacking** | Add favorable voters to district |
| **Hijacking** | Force two incumbents into one district |

### Redistricting Process

| Step | Description |
|------|-------------|
| **Census** | Every 10 years |
| **Reapportionment** | Assign seats to states |
| **Redistricting** | States draw new lines |
| **Who draws?** | State legislatures (usually) |
| **Alternatives** | Independent commissions |
