## ÔøΩüìã Unit Overview

The three branches of government interact in complex ways, checking and balancing each other while making policy. This unit explores how Congress, the President, the Courts, and the bureaucracy exercise their powers and interact with each other. Understanding these dynamics is essential for understanding how American government functions in practice! ‚öñÔ∏è

### Essential Questions

| Question | Focus |
|----------|-------|
| How does Congress make laws? | Legislative process |
| What powers does the President have? | Formal and informal |
| How do courts interpret law? | Judicial review; precedent |
| How does the bureaucracy work? | Implementation; discretion |
| How do branches interact? | Checks; cooperation; conflict |

### Key Concepts

| Concept | Description |
|---------|-------------|
| **Bicameralism** | Two-house legislature |
| **Executive power** | Presidential authority |
| **Judicial review** | Court review of constitutionality |
| **Bureaucracy** | Administrative agencies |

## üèõÔ∏è Congress: Structure and Powers

### Bicameral Legislature

| Chamber | Members | Term | Requirements | Representation |
|---------|---------|------|--------------|----------------|
| **House** | 435 | 2 years | 25 yrs; 7 yrs citizen; resident | By population |
| **Senate** | 100 | 6 years | 30 yrs; 9 yrs citizen; resident | 2 per state |

### Reasons for Bicameralism

| Reason | Description |
|--------|-------------|
| **Great Compromise** | Balanced large/small states |
| **Check** | Each house checks the other |
| **Deliberation** | Senate as "cooling saucer" |
| **Different perspectives** | Local vs. statewide |

### Enumerated Powers of Congress (Article I, Section 8)

| Power | Examples |
|-------|----------|
| **Taxing** | Levy taxes, duties |
| **Spending** | Appropriate funds |
| **Commerce** | Regulate interstate commerce |
| **Money** | Coin money; regulate value |
| **War** | Declare war; raise armies |
| **Naturalization** | Citizenship rules |
| **Necessary and Proper** | Implied powers |

### Exclusive Powers

| House Only | Senate Only |
|------------|-------------|
| Revenue bills originate | Confirm appointments |
| Impeach officials | Try impeachments |
| Elect president if no majority | Ratify treaties (2/3) |
| | Elect VP if no majority |

### Necessary and Proper Clause

| Aspect | Description |
|--------|-------------|
| **Also called** | Elastic Clause |
| **Function** | Grants implied powers |
| **Interpretation** | Broad (McCulloch v. Maryland) |
| **Examples** | Create bank; draft soldiers |

## üìù Legislative Process

### How a Bill Becomes Law

| Step | Description |
|------|-------------|
| **1. Introduction** | Member introduces bill |
| **2. Committee** | Referred to committee |
| **3. Subcommittee** | Detailed review |
| **4. Committee action** | Markup; vote to report |
| **5. Floor debate** | Full chamber discussion |
| **6. Vote** | Passage by chamber |
| **7. Other chamber** | Repeat process |
| **8. Conference** | Reconcile differences |
| **9. Final passage** | Both chambers vote |
| **10. President** | Sign or veto |

### Committee System

| Type | Function |
|------|----------|
| **Standing** | Permanent; policy areas |
| **Select/Special** | Temporary; specific issues |
| **Joint** | Both chambers |
| **Conference** | Reconcile versions |

### Key Committees

| Committee | Role |
|-----------|------|
| **Ways and Means (House)** | Tax policy |
| **Appropriations** | Spending |
| **Rules (House)** | Controls floor debate |
| **Judiciary** | Courts; civil rights |
| **Armed Services** | Military |

### House vs. Senate Procedures

| Feature | House | Senate |
|---------|-------|--------|
| **Rules Committee** | Controls debate | No equivalent |
| **Debate limits** | Strict time limits | Unlimited (filibuster) |
| **Amendments** | Must be germane | Can be non-germane |
| **Filibuster** | None | Yes (60 votes to end) |

### Filibuster and Cloture

| Concept | Description |
|---------|-------------|
| **Filibuster** | Extended debate to delay/block |
| **Cloture** | Ends debate (60 votes) |
| **Nuclear option** | Simple majority for some votes |
| **Effect** | 60-vote threshold for most legislation |

### Legislation Outcomes

| Outcome | Description |
|---------|-------------|
| **Signed** | Becomes law |
| **Vetoed** | Returns to Congress |
| **Override** | 2/3 both chambers |
| **Pocket veto** | Unsigned; Congress adjourns |

## üèõÔ∏è The Presidency

### Constitutional Powers

| Power | Description |
|-------|-------------|
| **Commander in Chief** | Military leadership |
| **Treaties** | Negotiate (Senate ratifies) |
| **Appointments** | Judges; ambassadors; officials |
| **Pardons** | Federal offenses |
| **State of Union** | Report to Congress |
| **Veto** | Reject legislation |
| **Convene Congress** | Special sessions |

### Informal Powers

| Power | Description |
|-------|-------------|
| **Executive orders** | Direct agencies |
| **Executive agreements** | International deals without Senate |
| **Signing statements** | Interpretation of laws |
| **Bully pulpit** | Use media; public opinion |
| **Executive privilege** | Withhold information |
| **Party leader** | Influence party members |
| **Crisis manager** | Lead in emergencies |

### Executive Orders

| Feature | Description |
|---------|-------------|
| **Definition** | Presidential directive to agencies |
| **Authority** | Constitution; statutes |
| **Effect** | Force of law |
| **Limit** | Can be overturned by Congress or courts |
| **Examples** | Desegregating military; DACA |

### Treaties vs. Executive Agreements

| Treaty | Executive Agreement |
|--------|---------------------|
| Requires 2/3 Senate | No Senate approval |
| Formal commitment | May be less binding |
| Can't be unilaterally withdrawn | President can withdraw |
| Takes longer | Faster |

### Veto Power

| Type | Description |
|------|-------------|
| **Regular veto** | Return with objections |
| **Override** | 2/3 of both chambers |
| **Pocket veto** | No signature; Congress adjourns |
| **Threat** | Influences legislation |

## ‚öñÔ∏è The Judiciary

### Structure of Federal Courts

| Level | Courts | Judges |
|-------|--------|--------|
| **District** | 94 trial courts | ~670 judges |
| **Circuit/Appeals** | 13 Courts of Appeals | ~180 judges |
| **Supreme Court** | 1 | 9 justices |

### Judicial Selection

| Aspect | Process |
|--------|---------|
| **Nomination** | President nominates |
| **Confirmation** | Senate (simple majority) |
| **Tenure** | Life terms |
| **Removal** | Only by impeachment |

### Nomination Factors

| Factor | Consideration |
|--------|---------------|
| **Ideology** | Policy views |
| **Qualifications** | Experience; competence |
| **Diversity** | Race; gender; geography |
| **Age** | Younger = longer service |
| **Confirmability** | Senate composition |

### Judicial Review

| Aspect | Description |
|--------|-------------|
| **Definition** | Power to declare laws unconstitutional |
| **Origin** | Marbury v. Madison (1803) |
| **Effect** | Courts can strike down laws |
| **Scope** | Federal and state laws |

### Marbury v. Madison (1803)

| Issue | Ruling |
|-------|--------|
| **Question** | Could Marbury get commission? |
| **Ruling** | Yes, entitled; but Court can't order |
| **Reason** | Section of Judiciary Act unconstitutional |
| **Significance** | Established judicial review |

### Judicial Philosophies

| Philosophy | Description |
|------------|-------------|
| **Judicial restraint** | Defer to legislature; narrow rulings |
| **Judicial activism** | Active policy role; broader interpretation |
| **Originalism** | Original intent/meaning |
| **Living Constitution** | Adapt to modern times |

### Precedent and Stare Decisis

| Concept | Description |
|---------|-------------|
| **Precedent** | Past decisions guide future |
| **Stare decisis** | "Let the decision stand" |
| **Purpose** | Consistency; predictability |
| **Override** | Courts can overturn precedent |
