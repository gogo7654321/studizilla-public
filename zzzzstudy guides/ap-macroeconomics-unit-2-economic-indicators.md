## ğŸ’° Gross Domestic Product (GDP)

### Definition

| Concept | Description |
|---------|-------------|
| **GDP** | Total market value of all final goods and services produced within a country in a year |
| **Market value** | Use prices |
| **Final goods** | Not intermediate |
| **Within country** | Geographic boundary |
| **Time period** | Usually one year |

### What's Included

| Included | Excluded |
|----------|----------|
| Final goods | Intermediate goods |
| New production | Used goods |
| Market transactions | Non-market activities |
| Goods produced in US | Goods produced abroad by US firms |

### What's Excluded

| Exclusion | Reason |
|-----------|--------|
| **Intermediate goods** | Avoid double counting |
| **Used goods** | Already counted |
| **Financial transactions** | Not production |
| **Transfer payments** | Not production |
| **Household production** | Not in market |
| **Underground economy** | Not reported |

## ğŸ“Š Calculating GDP

### Expenditure Approach

| Formula | GDP = C + I + G + NX |
|---------|---------------------|
| **C** | Consumer spending |
| **I** | Gross private domestic investment |
| **G** | Government spending |
| **NX** | Net exports (X - M) |

### Components Detailed

| Component | Includes |
|-----------|----------|
| **Consumption (C)** | Durable goods; nondurable goods; services |
| **Investment (I)** | Business equipment; construction; inventory changes |
| **Government (G)** | Government purchases (NOT transfers) |
| **Net Exports (NX)** | Exports - Imports |

### What Is Investment?

| Included | NOT Included |
|----------|--------------|
| New capital goods | Financial assets (stocks, bonds) |
| New construction | Transfer payments |
| Inventory changes | Consumer durables |

### Income Approach

| Components | |
|------------|--|
| Wages | + |
| Rent | + |
| Interest | + |
| Profits | = GDP |

### Value Added

| Concept | Description |
|---------|-------------|
| **Definition** | Value of output - Value of inputs |
| **Purpose** | Avoid double counting |
| **Sum of value added** | = GDP |

## ğŸ“Š Types of Unemployment

### Frictional Unemployment

| Concept | Description |
|---------|-------------|
| **Definition** | Between jobs; new entrants |
| **Cause** | Job search; matching |
| **Duration** | Usually short-term |
| **Examples** | Recent graduate; job switcher |
| **Not a problem** | Normal part of economy |

### Structural Unemployment

| Concept | Description |
|---------|-------------|
| **Definition** | Skills don't match jobs |
| **Cause** | Technology; globalization |
| **Duration** | Long-term |
| **Examples** | Displaced factory workers |
| **Solutions** | Training; education |

### Cyclical Unemployment

| Concept | Description |
|---------|-------------|
| **Definition** | Due to recession |
| **Cause** | Decline in demand |
| **Duration** | Until recovery |
| **Target of policy** | Eliminate through stimulus |

### Seasonal Unemployment

| Concept | Description |
|---------|-------------|
| **Definition** | Due to seasonal patterns |
| **Examples** | Ski instructors in summer |
| **Not in official rate** | Seasonally adjusted |

### Summary

| Type | Cause | Solution |
|------|-------|----------|
| **Frictional** | Job search | Job information |
| **Structural** | Skill mismatch | Training |
| **Cyclical** | Recession | Fiscal/monetary policy |
| **Seasonal** | Seasons | Adjustment |

## ğŸ”º Inflation

### Definition

| Concept | Description |
|---------|-------------|
| **Inflation** | Sustained increase in price level |
| **Deflation** | Sustained decrease in price level |
| **Disinflation** | Slowing inflation rate |

### Measuring Inflation

| Index | Description |
|-------|-------------|
| **CPI** | Consumer Price Index |
| **GDP deflator** | All goods in GDP |
| **PPI** | Producer prices |

### Consumer Price Index (CPI)

| Concept | Description |
|---------|-------------|
| **Definition** | Price of market basket |
| **Base year** | = 100 |
| **Calculated by** | Bureau of Labor Statistics |
| **Most common** | Inflation measure |

### CPI Formula

| Formula | |
|---------|--|
| **CPI** | = (Cost of basket this year / Cost in base year) Ã— 100 |
| **Inflation rate** | = [(CPIâ‚‚ - CPIâ‚) / CPIâ‚] Ã— 100 |

### CPI Problems

## ğŸ’¸ Costs of Inflation

### Anticipated Inflation

| Cost | Description |
|------|-------------|
| **Menu costs** | Changing prices |
| **Shoe leather costs** | Managing cash |
| **Tax distortions** | Bracket creep |

### Unanticipated Inflation

| Cost | Description |
|------|-------------|
| **Redistribution** | Borrowers gain; lenders lose |
| **Uncertainty** | Hard to plan |
| **Arbitrary** | Unfair transfers |

### Winners and Losers

| If Inflation Is Higher Than Expected | Effect |
|--------------------------------------|--------|
| **Borrowers** | Win (repay cheaper dollars) |
| **Lenders** | Lose (receive cheaper dollars) |
| **Fixed-income** | Lose (purchasing power falls) |
| **Debtors** | Win |
| **Creditors** | Lose |

### Real vs. Nominal Interest

| Concept | Description |
|---------|-------------|
| **Nominal rate** | Stated rate |
| **Real rate** | Adjusted for inflation |
| **Formula** | Real = Nominal - Inflation |

### Fisher Equation

| Formula | Nominal = Real + Expected Inflation |
|---------|-------------------------------------|
| **Real rate** | = Nominal - Inflation |
| **If expected** | Markets adjust |

### Question 3: Inflation Winners and Losers

**Assume the expected inflation rate was 2%, but actual inflation turned out to be 5%.**

**(a)** Explain whether borrowers or lenders benefited, and why.

**(b)** What happens to workers with fixed-wage contracts?

**(c)** Using the Fisher equation, if the nominal interest rate was 6%, what was the expected real rate vs. actual real rate?

### Model Answer 3:

**(a)** **Borrowers benefited.** They repay loans with dollars that have less purchasing power than expected. If a borrower took a loan expecting 2% inflation, they expected to repay in dollars worth 98% of original value; instead, dollars are worth only 95%, making debt easier to repay in real terms.

**(b)** **Workers with fixed wages are hurt.** Their nominal wages stay the same, but prices rose faster than expected. Real wages = Nominal wages / Price level. As price level increases more than expected, purchasing power (real wages) falls.

**(c)** Fisher Equation calculations:
```
Expected real rate = Nominal - Expected inflation = 6% - 2% = 4%
Actual real rate = Nominal - Actual inflation = 6% - 5% = 1%
```
Lenders expected to earn 4% real return but only earned 1%.
