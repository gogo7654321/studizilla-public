# ‚úèÔ∏è AP Drawing Portfolio Guide

## üìö Course Overview

AP Drawing is a portfolio-based course focusing on drawing concepts, mark-making, and the use of drawing materials and processes to communicate visual ideas.

---

## üéØ Portfolio Components

### Three Sections

```
AP DRAWING PORTFOLIO STRUCTURE:

1. SUSTAINED INVESTIGATION
   - 15 digital images
   - Written statement (1,200 characters)
   - Inquiry-based exploration
   - Shows growth in drawing
   - 60% of score

2. SELECTED WORKS
   - 5 works
   - 2 images each (10 total images)
   - Shows mastery of drawing
   - 40% of score

3. PHYSICAL PORTFOLIO
   - May be requested for validation
   - Keep original drawings accessible

TOTAL: 25 images submitted digitally
```

### What Makes Drawing Unique

```
DRAWING DEFINED:

Drawing includes works that emphasize:
- Line quality and variety
- Mark-making
- Surface manipulation
- Rendering techniques
- Use of drawing media

ACCEPTABLE MEDIA:
- Graphite, charcoal, colored pencil
- Ink, pastel, crayon
- Mixed drawing media
- Digital drawing
- Printmaking (when mark-based)

DRAWING VS. 2-D DESIGN:
Drawing emphasizes mark-making
2-D emphasizes design principles
(Some overlap is fine)
```

## üìã Sustained Investigation

### Choosing Your Inquiry

```
STRONG DRAWING INQUIRY QUESTIONS:

MARK-BASED:
"How does the quality of line
 convey emotional intensity?"

OBSERVATIONAL:
"How can I capture fleeting
 moments through gesture?"

CONCEPTUAL:
"How do accumulated marks
 build meaning over time?"

MATERIAL:
"What happens when traditional
 and digital drawing merge?"

PROCESS:
"How does erasure function
 as a drawing tool?"
```

### Investigation Development

```
DEVELOPMENT PHASES:

PHASE 1: EXPLORATION (Images 1-4)
- Mark-making experiments
- Media exploration
- Subject matter testing
- Finding direction

PHASE 2: FOCUS (Images 5-8)
- Narrowing inquiry
- Technique refinement
- Deeper exploration
- Building connections

PHASE 3: DEVELOPMENT (Images 9-12)
- Complex problem-solving
- Scale/scope expansion
- Risk-taking
- Synthesis emerging

PHASE 4: RESOLUTION (Images 13-15)
- Sophisticated solutions
- Mastery of approach
- Culminating works
- Investigation complete
```

### Photographing Drawings

```
PHOTOGRAPHY TIPS:

LIGHTING:
- Even, diffused light
- No hot spots
- Minimize shadow
- Accurate value capture

CAMERA POSITION:
- Parallel to work
- No keystoning
- Proper cropping
- Full frame

FOR GRAPHITE/CHARCOAL:
- Avoid glare
- Polarizing filter helpful
- Angle light carefully
- True value reproduction

FOR COLOR MEDIA:
- Accurate color balance
- Consistent white balance
- No color casts
- True saturation
```

## üé® Drawing Media

### Dry Media

```
GRAPHITE:
- Range of hardness (9H-9B)
- Smooth, controllable
- Erasable
- Can be precise or gestural

CHARCOAL:
- Vine, compressed, pencil
- Rich blacks
- Expressive marks
- Subtractive possibilities

COLORED PENCIL:
- Wax or oil-based
- Layering/blending
- Burnishing
- Color mixing

PASTEL:
- Soft, hard, oil, pastel pencil
- Rich color
- Blending capability
- Delicate or bold

CONTE:
- Earth tones primarily
- Sanguine, sepia, white, black
- Figure drawing tradition
- Toned paper work

CRAYON:
- Wax-based
- Resist techniques
- Bold marks
- Textural effects
```

### Wet Media

```
INK:
- Pen and brush
- Wash techniques
- Line and value
- Wet into wet

BRUSH DRAWING:
- Eastern traditions
- Expressive stroke
- Value variation
- Gesture capture

MIXED WET/DRY:
- Ink wash + charcoal
- Watercolor + colored pencil
- Layered approaches
- Combined effects
```
