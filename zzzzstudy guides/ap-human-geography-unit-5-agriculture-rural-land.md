## üöú Types of Agriculture

### Subsistence vs. Commercial

| Aspect | Subsistence | Commercial |
|--------|-------------|------------|
| **Purpose** | Feed family | Sell for profit |
| **Scale** | Small | Large |
| **Labor** | Intensive | Often mechanized |
| **Location** | LDCs | MDCs (primarily) |
| **Surplus** | Little or none | Goal is surplus |

### Subsistence Agriculture Types

| Type | Description | Location |
|------|-------------|----------|
| **Shifting cultivation** | Slash-and-burn; rotate fields | Tropics |
| **Pastoral nomadism** | Herding; follow resources | Arid regions |
| **Intensive subsistence** | High labor input; small plots | South/East Asia |

### Shifting Cultivation

| Aspect | Description |
|--------|-------------|
| **Also called** | Slash-and-burn; swidden |
| **Method** | Clear forest; farm; move on |
| **Cycle** | Return after soil recovers |
| **Location** | Amazon; Southeast Asia; Africa |
| **Crops** | Varied; root crops; grains |
| **Sustainability** | Works with low population density |

### Pastoral Nomadism

| Aspect | Description |
|--------|-------------|
| **Definition** | Herding animals; following resources |
| **Animals** | Camels; goats; sheep; cattle |
| **Location** | Central Asia; Middle East; Africa |
| **Movement** | Seasonal; based on water and pasture |
| **Challenges** | Climate change; borders; modernization |

### Intensive Subsistence Agriculture

| Type | Description | Location |
|------|-------------|----------|
| **Wet rice** | Paddy cultivation | Monsoon Asia |
| **Non-rice** | Other grains; vegetables | Drier areas of Asia |

### Commercial Agriculture Types

| Type | Description | Example |
|------|-------------|---------|
| **Mixed crop and livestock** | Both crops and animals | Corn Belt USA |
| **Dairy** | Milk production | Wisconsin; New Zealand |
| **Grain farming** | Large-scale grain crops | Great Plains |
| **Livestock ranching** | Extensive cattle/sheep | Argentina; Australia |
| **Mediterranean** | Olives; grapes; citrus | California; Mediterranean |
| **Commercial gardening** | Fruits; vegetables; flowers | Near cities |
| **Plantation** | Single cash crop; export | Tropics (coffee, bananas) |

| Aspect | Description |

| **Created** | 1826 by Johann Heinrich von Th√ºnen |
| **Purpose** | Explains agricultural land use |
| **Concept** | Land use varies by distance from market |

### Von Th√ºnen Rings

| Ring | Land Use | Reason |
|------|----------|--------|
| **1 (center)** | Market gardening; dairy | Perishable; high value |
| **2** | Forest | Fuel wood; heavy |

| Assumption | Reality |
|------------|---------|

## üß¨ Agricultural Technology Comparison

### Traditional vs. Modern Agriculture

| Aspect | Traditional | Modern/Industrial |
|--------|-------------|-------------------|
| **Seeds** | Saved from harvest; local varieties | Purchased; hybrid or GMO |
| **Fertilizer** | Animal manure; crop rotation | Synthetic (nitrogen, phosphorus) |
| **Pest control** | Manual; natural predators | Chemical pesticides |
| **Labor** | Family; intensive | Machines; minimal workers |
| **Scale** | Small plots | Large commercial operations |
| **Diversity** | Polyculture; many varieties | Monoculture; few varieties |
| **Market** | Local consumption | Global export |

### Organic vs. Conventional Comparison

| Factor | Organic | Conventional |
|--------|---------|--------------|
| **Synthetic chemicals** | Prohibited | Standard practice |
| **Yields** | 10-20% lower typically | Higher (with inputs) |
| **Price to consumer** | 20-100% higher | Standard |
| **Environmental impact** | Lower chemical pollution | Higher chemical use |
| **Labor intensity** | Higher | Lower |
| **Certification** | Required; inspected | Not required |
