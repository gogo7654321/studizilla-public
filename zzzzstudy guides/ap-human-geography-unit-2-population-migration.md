## üåç Population Distribution

### Global Distribution Patterns

| Region | Population Characteristics |
|--------|---------------------------|
| **East Asia** | ~25% of world; China, Japan, Koreas |
| **South Asia** | ~25% of world; India, Pakistan, Bangladesh |
| **Europe** | Dense; declining growth |
| **North America** | Concentrated on coasts/Great Lakes |
| **Africa** | Fastest growing continent |

### Factors Affecting Distribution

| Factor | Favorable | Unfavorable |
|--------|-----------|-------------|
| **Climate** | Temperate; moderate | Extreme cold/heat; desert |
| **Terrain** | Flat; coastal | Mountains; swamps |
| **Water** | Rivers; lakes | Arid regions |
| **Soil** | Fertile | Poor quality |
| **Resources** | Minerals; energy | Scarce resources |
| **Accessibility** | Transportation routes | Isolated areas |

### Ecumene

| Concept | Description |
|---------|-------------|
| **Definition** | Permanently inhabited areas |
| **Percentage** | ~35-40% of Earth's land |
| **Expansion** | Technology extends habitable areas |
| **Non-ecumene** | Antarctica; Sahara; Arctic |

## üìä Demographic Transition Model (DTM)

### Overview

| Aspect | Description |
|--------|-------------|
| **Purpose** | Explains population change over time |
| **Stages** | 4 (sometimes 5) stages |
| **Based on** | Historical European experience |
| **Limitation** | May not apply everywhere |

### Stage 1: Pre-Industrial

| Characteristic | Description |
|----------------|-------------|
| **Birth rate** | Very high |
| **Death rate** | Very high (fluctuates) |
| **Population** | Stable; low |
| **Causes** | Disease; famine; lack of medicine |
| **Today** | No countries at this stage |

### Stage 2: Early Transition

| Characteristic | Description |
|----------------|-------------|
| **Birth rate** | Still high |
| **Death rate** | Rapidly declining |
| **Population** | Rapid growth |
| **Causes** | Improved healthcare; sanitation; food |
| **Examples** | Afghanistan; Niger; many African nations |

### Stage 3: Late Transition

| Characteristic | Description |
|----------------|-------------|
| **Birth rate** | Falling |
| **Death rate** | Low |
| **Population** | Slowing growth |
| **Causes** | Urbanization; women's education; contraception |
| **Examples** | India; Mexico; Brazil |

### Stage 4: Post-Industrial

| Characteristic | Description |
|----------------|-------------|
| **Birth rate** | Low |
| **Death rate** | Low |
| **Population** | Stable or slow growth |
| **Causes** | High living standards; education |
| **Examples** | USA; UK; France; Australia |

### Stage 5: Decline (Proposed)

| Characteristic | Description |
|----------------|-------------|
| **Birth rate** | Very low (below replacement) |
| **Death rate** | Low but rising (aging) |
| **Population** | Declining |
| **Examples** | Japan; Germany; Italy; Russia |

### DTM Summary Table

| Stage | Birth Rate | Death Rate | NIR | Pop Growth |
|-------|------------|------------|-----|------------|
| 1 | High | High | Low | Stable |
| 2 | High | Falling | High | Rapid growth |
| 3 | Falling | Low | Moderate | Slowing growth |
| 4 | Low | Low | Low | Stable |
| 5 | Very low | Low/rising | Negative | Decline |

## üåê Population Theories

### Malthusian Theory

| Aspect | Description |
|--------|-------------|
| **Theorist** | Thomas Malthus (1798) |
| **Claim** | Population grows faster than food supply |
| **Population** | Grows exponentially (2, 4, 8, 16...) |
| **Food** | Grows arithmetically (1, 2, 3, 4...) |
| **Checks** | Famine; disease; war |
| **Criticism** | Didn't predict technology |

### Neo-Malthusian View

| Aspect | Description |
|--------|-------------|
| **Modern version** | Concern about resource limits |
| **Focus** | Environment; sustainability |
| **Advocates** | Paul Ehrlich (Population Bomb) |
| **Solutions** | Birth control; consumption limits |

### Anti-Malthusian Views

| View | Description |
|------|-------------|
| **Esther Boserup** | Population growth drives innovation |
| **Julian Simon** | People are the "ultimate resource" |
| **Cornucopians** | Technology will solve problems |

### Epidemiological Transition Model

| Stage | Causes of Death |
|-------|-----------------|
| 1 | Infectious diseases; famine |
| 2 | Receding pandemics |
| 3 | Degenerative diseases (heart; cancer) |
| 4 | Delayed degenerative; lifestyle |
| 5 | Reemergence of infectious (HIV; new diseases) |

## üö∂ Migration

### Types of Migration

| Type | Definition | Example |
|------|------------|---------|
| **Internal** | Within a country | Rural to urban |
| **International** | Between countries | Mexico to USA |
| **Voluntary** | By choice | Economic opportunity |
| **Forced** | No choice | Refugees; slavery |
| **Step migration** | Series of moves | Village‚Üítown‚Üícity |
| **Chain migration** | Following family/friends | Communities abroad |
| **Transhumance** | Seasonal movement | Herders with livestock |

### Push and Pull Factors

| Push Factors | Pull Factors |
|--------------|--------------|
| War; conflict | Peace; stability |
| Poverty | Economic opportunity |
| Natural disasters | Good climate |
| Persecution | Freedom; rights |
| Unemployment | Jobs |
| Overpopulation | Space; resources |
| Environmental degradation | Clean environment |

### Ravenstein's Laws of Migration

| Law | Description |
|-----|-------------|
| Most migrants move short distances | Distance decay |
| Long-distance migrants go to big cities | Urban attraction |
| Urban residents migrate less | Already in opportunity centers |
| Females migrate more within country | Short distance |
| Males migrate more internationally | Longer distances |
| Most migrants are adults | Working age |
| Migration flows produce counter-flows | Return migration |
| Migration increases with development | More mobility |

### Gravity Model

| Aspect | Description |
|--------|-------------|
| **Formula** | Interaction = (Pop‚ÇÅ √ó Pop‚ÇÇ) √∑ Distance¬≤ |
| **Meaning** | Larger, closer places interact more |
| **Application** | Predict migration flows |
| **Factors** | Size matters; distance reduces flow |

## üìä Migration Effects

| Effect | Description |
|--------|-------------|
| **Remittances** | Money sent home |
| **Brain drain** | Loss of educated workers |
| **Reduced unemployment** | Fewer workers seeking jobs |

### Effects on Destination Countries

| Effect | Description |
|--------|-------------|
| **Labor** | Fill workforce needs |
| **Cultural diversity** | New ideas; foods; traditions |
| **Population growth** | Counter aging |
| **Challenges** | Integration; services |

### Remittances

| Aspect | Description |
|--------|-------------|
| **Definition** | Money migrants send to home country |
