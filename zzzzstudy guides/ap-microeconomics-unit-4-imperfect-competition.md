### Characteristics

| Feature | Description |
|---------|-------------|

### Barriers to Entry

| Type | Description |
|------|-------------|
| **Economies of scale** | Natural monopoly |

### Natural Monopoly

| Concept | Description |
|---------|-------------|
| **Definition** | One firm can serve market at lower cost |
| **LRAC** | Falls over entire relevant range |
| **Examples** | Utilities, railways |
| **Problem** | Competition would be wasteful |
| **Solution** | Regulation |

| **Luxottica (eyewear)** | Economies of scale | Near-monopoly (~80% of brands) |

| Concept | Description |
|---------|-------------|
| **P > MC** | Price exceeds marginal cost |
| **Underproduction** | Less than socially optimal |

### Deadweight Loss

| Definition | Lost total surplus |
|------------|-------------------|
| **Cause** | Q less than competitive |
| **Triangle** | Between D, MC, from Qm to Qc |

### Welfare Effects

| Surplus | Effect |
|---------|--------|
| **CS** | Decreases (higher P, lower Q) |

| **Total surplus** | Decreases (DWL) |

### Transfer vs. Loss

| Effect | Description |
|--------|-------------|
| **Transfer** | Some CS becomes PS |
| **Net loss** | DWL (not transferred, just lost) |

## ğŸ’° Price Discrimination

### Definition

| Concept | Description |
|---------|-------------|
| **Price discrimination** | Charging different prices to different buyers |
| **Same product** | Different prices |
| **Requirement** | Market power |
| **Requirement** | Can separate markets |
| **Requirement** | No resale possible |

### Degrees of Price Discrimination

| Degree | Description | Example |
|--------|-------------|---------|
| **First degree** | Each buyer pays WTP | Haggling |
| **Second degree** | Different quantities | Bulk discounts |
| **Third degree** | Different groups | Student discounts |

### First Degree (Perfect) Price Discrimination

| Feature | Description |
|---------|-------------|
| **Captures all CS** | Every buyer pays max WTP |
| **No DWL** | Produces competitive quantity |
| **All surplus to producer** | Maximum PS |
| **Allocatively efficient** | P = MC for marginal unit |

### Third Degree Price Discrimination

| Feature | Description |
|---------|-------------|
| **Segment markets** | Different elasticities |
| **Higher price** | To inelastic group |
| **Lower price** | To elastic group |
| **Increases profit** | Beyond single price |

### Price Discrimination and Elasticity

| Group | Elasticity | Price |
|-------|------------|-------|
| **Business travelers** | Low (inelastic) | High |
| **Leisure travelers** | High (elastic) | Low |

### Real-World Price Discrimination Examples

| Type | Example | How It Works |
|------|---------|--------------|
| **1st Degree** | Car dealer negotiation | Each buyer pays different price |
| **2nd Degree** | Costco bulk pricing | Lower per-unit price for more |
| **3rd Degree** | Movie theater tickets | Senior, student, child discounts |
| **3rd Degree** | Airline tickets | Business vs. leisure fares |
| **3rd Degree** | Software pricing | Student editions at lower price |
| **Peak pricing** | Uber surge pricing | Higher price during high demand |

## ğŸ® Game Theory

### Basic Concepts

| Term | Definition |
|------|------------|
| **Game** | Strategic interaction |
| **Players** | Decision makers |
| **Strategies** | Possible actions |
| **Payoffs** | Outcomes from strategies |
| **Payoff matrix** | Shows all outcomes |

### Dominant Strategy

| Concept | Description |
|---------|-------------|
| **Definition** | Best choice regardless of rival's action |
| **Always choose** | Dominant strategy if exists |
| **May not exist** | Not all games have one |

### Nash Equilibrium

| Concept | Description |
|---------|-------------|
| **Definition** | No player wants to change strategy |
| **Stable** | Given others' strategies |
| **May not be optimal** | For group |

### Prisoners' Dilemma

| Concept | Description |
|---------|-------------|
| **Classic game** | Cooperation vs. defection |
| **Dominant strategy** | Both defect |
| **Nash equilibrium** | Worse than cooperation |
| **Lesson** | Individual rationality â‰  group optimality |

### Prisoners' Dilemma Example

|  | Firm B: Cooperate | Firm B: Defect |
|--|-------------------|----------------|
| **Firm A: Cooperate** | (10, 10) | (0, 15) |
| **Firm A: Defect** | (15, 0) | (5, 5) |

| Analysis | Result |
|----------|--------|
| **Dominant strategy** | Both defect |
| **Nash equilibrium** | (5, 5) |
| **Better outcome** | (10, 10) if cooperate |
| **Problem** | Each tempted to defect |

### How to Solve Game Theory Problems (Step-by-Step)

| Step | Action |
|------|--------|
| **1** | Draw the payoff matrix |
| **2** | For each player, find best response to each opponent strategy |

### Collusion

| Concept | Description |
|---------|-------------|
| **Definition** | Firms cooperate to act as monopoly |
| **Goal** | Maximize joint profit |
| **Illegal** | In most countries |
| **Examples** | Price fixing, output limits |

### Cartel

| Concept | Description |
|---------|-------------|
| **Definition** | Formal agreement to collude |
| **Famous example** | OPEC |
| **Behavior** | Set monopoly price |
| **Instability** | Incentive to cheat |

### Why Cartels Are Unstable

| Problem | Description |
|---------|-------------|
| **Incentive to cheat** | Gain by secretly lowering price |
| **Prisoners' dilemma** | Each better off defecting |
| **Detection** | Hard to monitor |
| **Enforcement** | Can't legally enforce |

### Factors Favoring Collusion

| Factor | Why |
|--------|-----|
| **Few firms** | Easier to coordinate |
| **Similar costs** | Less disagreement |
| **High barriers** | Can't be undercut |
| **Homogeneous product** | Easier to monitor |
| **Frequent interaction** | Repeated games |
