## üó≥Ô∏è Electoral Systems

### Electoral System Types

| Type | Description | Example |
|------|-------------|---------|
| **Plurality/Majoritarian** | Most votes wins | UK, Nigeria |
| **Proportional (PR)** | Seats match vote share | None of 6 (Russia partial) |
| **Mixed** | Combination | Russia, Mexico |

### Plurality/Majoritarian

| Feature | Description |
|---------|-------------|
| **FPTP** | First Past the Post |
| **Single-member districts** | One rep per district |
| **Winner-take-all** | Most votes wins |
| **Effect** | Favors large parties |
| **Used by** | UK; Nigeria |

### Proportional Representation

| Feature | Description |
|---------|-------------|
| **Multi-member districts** | Several reps per district |
| **Vote share = seat share** | Proportional |
| **Party lists** | Voters choose party |
| **Threshold** | Minimum to get seats |
| **Effect** | More parties represented |

### Mixed Systems

| Feature | Description |
|---------|-------------|
| **Combines** | SMD + PR |
| **Two votes** | Candidate + party |
| **Effect** | Balance local and proportional |
| **Russia** | 225 SMD + 225 PR |
| **Mexico** | 300 SMD + 200 PR |

### Electoral System Effects

| Effect | Description |
|--------|-------------|
| **Duverger's Law** | Plurality ‚Üí two parties |
| **Proportionality** | PR ‚Üí more parties |
| **Representation** | PR = more minorities |
| **Stability** | Plurality = clearer majorities |
