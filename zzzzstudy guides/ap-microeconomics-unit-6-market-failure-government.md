## üìâ Negative Externalities

### Negative Externality in Production

| Concept | Description |
|---------|-------------|
| **Example** | Factory pollutes air |
| **Private cost** | Cost to firm |
| **External cost** | Cost to society |
| **Social cost** | Private + external |

### Graphical Analysis

| Curve | Description |
|-------|-------------|
| **Supply (private)** | Marginal private cost (MPC) |
| **Supply (social)** | Marginal social cost (MSC) |
| **MSC above MPC** | By external cost |

### Market Outcome

| Result | Description |
|--------|-------------|
| **Overproduction** | Q market > Q optimal |
| **Price too low** | Doesn't include external cost |
| **Deadweight loss** | Triangle between curves |

### Correcting Negative Externalities

| Solution | Description |
|----------|-------------|
| **Tax** | Shift supply up to MSC |
| **Regulation** | Limit quantity |
| **Cap and trade** | Tradable permits |
| **Property rights** | Coase theorem |

### Pigouvian Tax

| Concept | Description |
|---------|-------------|
| **Definition** | Tax equal to external cost |
| **Effect** | Supply shifts to MSC |
| **Result** | Socially optimal quantity |
| **Internalizes** | The externality |

## üìú Coase Theorem

### Statement

| Concept | Description |
|---------|-------------|
| **Coase theorem** | Private parties can solve externalities |
| **Conditions** | Property rights defined; low transaction costs |
| **Outcome** | Efficient regardless of initial assignment |

### Requirements

| Requirement | Description |
|-------------|-------------|
| **Clear property rights** | Someone owns right |
| **Low transaction costs** | Easy to bargain |
| **Few parties** | Easier negotiation |

### Example

| Scenario | Outcome |
|----------|---------|
| **Factory pollutes; residents have right** | Factory pays to pollute or reduces |
| **Factory has right to pollute** | Residents pay factory to reduce |
| **Either way** | Efficient outcome if can bargain |

### Limitations

| Problem | Description |
|---------|-------------|
| **High transaction costs** | Many parties; hard to organize |
| **Unclear property rights** | Who owns clean air? |
| **Free rider problem** | Won't contribute |
| **Strategic behavior** | Hold out for more |

## üèõÔ∏è Public Goods

### Characteristics of Goods

| Property | Definition |
|----------|------------|
| **Excludable** | Can prevent use by non-payers |
| **Rival** | One person's use reduces another's |

### Types of Goods

| Type | Excludable | Rival | Examples |
|------|------------|-------|----------|
| **Private good** | Yes | Yes | Food, clothing |
| **Public good** | No | No | National defense, fireworks |
| **Common resource** | No | Yes | Fish in ocean, clean air |
| **Club good** | Yes | No | Cable TV, gym |

### Public Goods

| Concept | Description |
|---------|-------------|
| **Non-excludable** | Can't prevent free riders |
| **Non-rival** | One's use doesn't reduce another's |
| **Problem** | Markets underproduce |
| **Solution** | Government provision |

### Free Rider Problem

| Concept | Description |
|---------|-------------|
| **Definition** | Benefit without paying |
| **Why** | Non-excludable |
| **Result** | Little private production |
| **Solution** | Tax-funded government provision |

### Examples of Public Goods

| Good | Why Public |
|------|------------|
| **National defense** | Can't exclude citizens |
| **Street lighting** | Non-rival; non-excludable |
| **Basic research** | Benefits all; hard to charge |
| **Clean air** | Non-excludable |

## ‚öñÔ∏è Coase Theorem in Practice

### Conditions for Coase Theorem

| Condition | Why Needed |
|-----------|------------|
| **Low transaction costs** | Negotiation must be cheap |
| **Clear property rights** | Someone must own the right |
| **Perfect information** | Both parties know costs/benefits |
| **Few parties** | Many parties = coordination problems |

### Coase Theorem Example

**Scenario**: Factory pollutes, affecting 10 nearby homes
- Pollution damage: $1,000 per home = $10,000 total
- Abatement cost for factory: $6,000

| Rights Assignment | Outcome |
|-------------------|---------|
| **Factory has right to pollute** | Homeowners pay factory $6,000-$10,000 to stop |
| **Homeowners have right to clean air** | Factory pays $6,000 to abate |
| **Either way** | Factory stops polluting (efficient outcome) |

### Real-World Complications

| Problem | Example |
|---------|---------|
| **High transaction costs** | Lawyers, time, uncertainty |
| **Holdout problem** | One party refuses to bargain |
| **Many parties** | Air pollution affects millions |
| **Unequal bargaining power** | Poor residents vs. wealthy corporation |

## üéØ Extended FRQ Practice

### Problem 1: Complete Externality Analysis

A factory produces steel and also produces pollution. The private supply is P = 10 + Q, the demand is P = 50 - Q, and the external cost is $8 per unit.

(a) Find market equilibrium price and quantity
(b) Find socially optimal price and quantity
(c) Calculate deadweight loss at market equilibrium
(d) What tax would achieve the optimal outcome?

**Solutions**:
(a) Market: 10 + Q = 50 - Q ‚Üí 2Q = 40 ‚Üí Q = 20, P = $30

(b) Social supply: P = 18 + Q (add $8 external cost)
    18 + Q = 50 - Q ‚Üí 2Q = 32 ‚Üí Q = 16
    P = 50 - 16 = $34

(c) DWL = 0.5 √ó (Qm - Qs) √ó External Cost
    DWL = 0.5 √ó (20 - 16) √ó $8 = $16

(d) Tax = $8 per unit (equals external cost)

### Problem 2: Public Goods

Two consumers, A and B, value fireworks:
- A: P = 20 - Q
- B: P = 30 - Q
- MC = $35

Find optimal quantity.

**Solution**:
Vertical sum: P = (20 - Q) + (30 - Q) = 50 - 2Q
Set equal to MC: 50 - 2Q = 35
2Q = 15, Q* = 7.5 units
