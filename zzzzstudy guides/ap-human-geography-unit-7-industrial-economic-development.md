## üìç Industrial Location Theory

### Weber's Least Cost Theory

| Concept | Description |
|---------|-------------|
| **Created** | Alfred Weber (1909) |
| **Goal** | Minimize production costs |
| **Factors** | Transportation; labor; agglomeration |

### Weber's Factors

| Factor | Description |
|--------|-------------|
| **Transportation** | Cost to move materials and products |
| **Labor** | Wage costs |
| **Agglomeration** | Benefits of clustering |
| **Deglomeration** | Costs of overcrowding |

### Locational Triangle

| Input | Consideration |
|-------|---------------|
| **Raw materials** | Weight and location |
| **Market** | Where products sold |
| **Labor** | Where workers available |
| **Optimal location** | Minimizes total transport cost |

### Weight-Losing vs. Weight-Gaining

| Type | Locate Near | Examples |
|------|-------------|----------|
| **Weight-losing** | Raw materials | Steel; copper |
| **Weight-gaining** | Market | Beverages; bread |
| **Footloose** | Flexible | Electronics; software |

### Other Location Factors

| Factor | Description |
|--------|-------------|
| **Energy** | Power availability |
| **Capital** | Investment access |
| **Infrastructure** | Transportation; utilities |
| **Government** | Policies; incentives |
| **Quality of life** | Attract workers |

## üìà Measuring Development

### Economic Indicators

| Indicator | Description |
|-----------|-------------|
| **GDP (Gross Domestic Product)** | Total output of goods and services |
| **GDP per capita** | GDP divided by population |
| **GNI (Gross National Income)** | Income earned by residents |
| **GNI per capita (PPP)** | Adjusted for purchasing power |

### Social Indicators

| Indicator | Description |
|-----------|-------------|
| **Literacy rate** | % who can read/write |
| **Life expectancy** | Average years of life |
| **Infant mortality rate** | Deaths under 1 per 1,000 births |
| **Access to healthcare** | Medical services availability |
| **Access to education** | Schooling availability |

### Composite Indicators

| Indicator | Components |
|-----------|------------|
| **Human Development Index (HDI)** | Life expectancy, education, income |
| **Gender Inequality Index (GII)** | Reproductive health, empowerment, labor |
| **Gender Development Index (GDI)** | HDI adjusted for gender |
| **Inequality-adjusted HDI (IHDI)** | HDI adjusted for inequality |

### Human Development Index (HDI)

| Component | Measure |
|-----------|---------|
| **Health** | Life expectancy at birth |
| **Education** | Mean and expected years of schooling |
| **Income** | GNI per capita (PPP) |
| **Scale** | 0 to 1 (higher = more developed) |

### Development Categories

| Category | HDI Range | Examples |
|----------|-----------|----------|
| **Very High** | 0.800+ | Norway, USA, Germany |
| **High** | 0.700-0.799 | Mexico, China, Brazil |
| **Medium** | 0.550-0.699 | India, Ghana |
| **Low** | Below 0.550 | Niger, Chad, CAR |

## üåê Development Theories

### Rostow's Modernization Theory

| Stage | Description |
|-------|-------------|
| **1. Traditional society** | Subsistence agriculture |
| **2. Preconditions for takeoff** | Infrastructure; banking |
| **3. Takeoff** | Rapid industrialization |
| **4. Drive to maturity** | Diversification |
| **5. Age of mass consumption** | Consumer goods; services |

### Rostow Model Critiques

| Critique | Description |
|----------|-------------|
| **Eurocentric** | Based on Western experience |
| **Linear assumption** | Development not always sequential |
| **Ignores exploitation** | Colonial legacies |
| **Ignores context** | Different challenges today |

### Dependency Theory

| Concept | Description |
|---------|-------------|
| **Core-periphery** | Rich nations exploit poor |
| **Colonial legacy** | Created unequal relationships |
| **Perpetuates poverty** | Structures maintain inequality |
| **Solution** | Break dependency; self-reliance |

### Wallerstein's World-Systems Theory

| Zone | Description |
|------|-------------|
| **Core** | Dominant economies; exploit periphery |
| **Semi-periphery** | Middle; both exploit and are exploited |
| **Periphery** | Exploited; raw material exporters |

### World-Systems Examples

| Zone | Countries |
|------|-----------|
| **Core** | USA, Western Europe, Japan |
| **Semi-periphery** | China, Brazil, India, Mexico |
| **Periphery** | Many African, Asian nations |

## üì¶ Development Strategies

### Self-Sufficiency Model

| Aspect | Description |
|--------|-------------|
| **Goal** | Reduce imports; domestic production |
| **Methods** | Tariffs; quotas; government investment |
| **Example** | India (pre-1990s) |
| **Outcome** | Slow growth; inefficiency |

### Export-Oriented Model

| Aspect | Description |
|--------|-------------|
| **Goal** | Grow through exports |
| **Methods** | Foreign investment; free trade zones |
| **Example** | Asian Tigers |
| **Outcome** | Rapid growth; but dependency |

| Concept | Description |

## üí∞ Development Strategies Comparison

### Self-Sufficiency vs. Export-Oriented

| Aspect | Self-Sufficiency (Import Substitution) | Export-Oriented (International Trade) |
|--------|----------------------------------------|---------------------------------------|
| **Goal** | Reduce imports; domestic production | Grow through exports |
| **Trade policy** | High tariffs; quotas | Low tariffs; free trade zones |
| **Government role** | Heavy intervention | Limited; incentives |
| **Industries** | Diverse; meet domestic needs | Specialized; comparative advantage |
| **Historical example** | India (1947-1991) | South Korea, Taiwan |
| **Advantages** | Self-reliance; balanced economy | Rapid growth; technology transfer |
| **Disadvantages** | Inefficiency; slow growth | Dependency; inequality |

### Development Success Stories

| Country | Strategy | Key Factors | Outcome |
|---------|----------|-------------|---------|
| **South Korea** | Export-led | Education, state investment, chaebol | High-income |
| **Singapore** | Trade hub + services | Strategic location, governance | Very high-income |
| **China** | Gradual market opening | SEZs, FDI, cheap labor | Rapid growth |
| **Botswana** | Resource management | Diamond wealth, good governance | Upper-middle income |
| **Vietnam** | Doi Moi reforms | Market socialism, FDI | Rapid growth |
