## ÔøΩüìã Unit Overview

Political institutions are the structures through which power is exercised. This unit examines the executive, legislative, and judicial branches across the six core countries, comparing how different systems organize and distribute power! üèõÔ∏è

### Essential Questions

| Question | Focus |
|----------|-------|
| How is executive power organized? | Presidents vs. prime ministers |
| How do legislatures function? | Lawmaking and representation |
| What role do courts play? | Judicial independence |
| How does bureaucracy work? | Implementation and power |
| How do institutions shape politics? | Comparative effects |

### Institutional Categories

| Institution | Function |
|-------------|----------|
| **Executive** | Implement policy; lead government |
| **Legislature** | Make laws; represent citizens |
| **Judiciary** | Interpret laws; resolve disputes |
| **Bureaucracy** | Administer programs |

## üëî Executive Systems

### Types of Executives

| Type | Description | Examples |
|------|-------------|----------|
| **Parliamentary** | PM from legislature | UK |
| **Presidential** | Separate election | Mexico, Nigeria |
| **Semi-presidential** | President + PM | Russia |
| **Theocratic** | Religious leader | Iran |
| **Party-state** | Party leader | China |

### Parliamentary Systems

| Feature | Description |
|---------|-------------|
| **Selection** | PM chosen by legislature |
| **Removal** | Vote of no confidence |
| **Cabinet** | From legislature |
| **Fusion of powers** | Executive = legislative majority |
| **Collective responsibility** | Cabinet solidarity |

### Presidential Systems

| Feature | Description |
|---------|-------------|
| **Selection** | Separate election |
| **Fixed term** | Cannot be removed easily |
| **Separation of powers** | Independent branches |
| **Cabinet** | Appointed by president |
| **Individual responsibility** | President accountable |

### Semi-Presidential Systems

| Feature | Description |
|---------|-------------|
| **Dual executive** | President + PM |
| **President** | Elected directly; foreign policy |
| **PM** | From legislature; domestic policy |
| **Cohabitation** | Different parties possible |
| **Example** | Russia (though dominated by president) |

| Power | Description |
|-------|-------------|

| Chamber | Features |
|---------|----------|

| Reality | Description |
|---------|-------------|

| **Rubber stamp** | Approves presidential agenda |

### China NPC

| Feature | Description |
|---------|-------------|
| **Size** | ~2,980 deputies |
| **Sessions** | Meets ~2 weeks annually |
| **Function** | Approve; not deliberate |

| Feature | Description |

| **Term** | 4 years |
| **Elections** | Candidates vetted by Guardian Council |
| **Powers** | Can question ministers |
| **Limits** | Guardian Council veto |

### Nigeria National Assembly
