## üå°Ô∏è Greenhouse Effect

### Natural Greenhouse Effect

| Component | Role |
|-----------|------|
| **Sun** | Provides energy as shortwave radiation |
| **Earth's surface** | Absorbs, re-emits as infrared |
| **Greenhouse gases** | Absorb and re-emit infrared |
| **Effect** | Warms atmosphere |
| **Importance** | Makes Earth habitable |

### Without Greenhouse Effect

| Condition | Value |
|-----------|-------|
| **Earth's temperature** | Would be ~-18¬∞C (0¬∞F) |
| **Current average** | ~15¬∞C (59¬∞F) |
| **Difference** | ~33¬∞C warming |

### Enhanced Greenhouse Effect

| Concept | Description |
|---------|-------------|
| **Human activities** | Increase greenhouse gases |
| **More heat trapped** | Additional warming |
| **Result** | Global warming |

## üî¥ Stratospheric Ozone Depletion

### Ozone Layer

| Characteristic | Description |
|----------------|-------------|
| **Location** | Stratosphere (15-35 km) |
| **Function** | Absorbs UV radiation |
| **Formation** | O‚ÇÇ + UV ‚Üí 2O; O + O‚ÇÇ ‚Üí O‚ÇÉ |
| **Protection** | Shields life from UV-B, UV-C |

### Ozone-Depleting Substances

| Substance | Use |
|-----------|-----|
| **CFCs** | Refrigerants, aerosols |
| **HCFCs** | Transitional replacement |
| **Halons** | Fire extinguishers |
| **Mechanism** | Cl/Br catalyzes O‚ÇÉ breakdown |

### Ozone Destruction

| Step | Process |
|------|---------|
| **1** | CFCs rise to stratosphere |
| **2** | UV breaks down CFCs |
| **3** | Free chlorine released |
| **4** | Cl + O‚ÇÉ ‚Üí ClO + O‚ÇÇ |
| **5** | ClO + O ‚Üí Cl + O‚ÇÇ |
| **6** | One Cl destroys thousands of O‚ÇÉ |

### Ozone Hole

| Characteristic | Description |
|----------------|-------------|
| **Location** | Antarctica (primarily) |
| **Timing** | September-October |
| **Why Antarctica?** | Polar stratospheric clouds |
| **Current status** | Slowly recovering |

### Effects of Ozone Depletion

| Effect | Description |
|--------|-------------|
| **Skin cancer** | Increased UV exposure |
| **Cataracts** | Eye damage |
| **Immune suppression** | Weakened immunity |
| **Crop damage** | Reduced yields |
| **Marine life** | Phytoplankton affected |

### Montreal Protocol

| Aspect | Description |
|--------|-------------|
| **Year** | 1987 |
| **Purpose** | Phase out ODS |
| **Success** | ODS declining |
