## üåç The Copernican Revolution

### Ptolemaic System (Geocentric)

| Feature | Description |
|---------|-------------|
| **Earth-centered** | Earth at center of universe |
| **Perfect spheres** | Crystalline spheres carry planets |
| **Epicycles** | Circular paths on circular paths |
| **Accepted** | Church-approved; biblical support |
| **Problems** | Couldn't accurately predict planetary positions |

### Nicolaus Copernicus (1473-1543)

| Contribution | Details |
|--------------|---------|
| **Heliocentric theory** | Sun at center |
| **Work** | *On the Revolutions of Heavenly Spheres* (1543) |
| **Published** | Year of his death |
| **Still used circles** | Not fully accurate |
| **Significance** | Started revolution in astronomy |

### Tycho Brahe (1546-1601)

| Contribution | Details |
|--------------|---------|

### Johannes Kepler (1571-1630)

| Law | Description |
|-----|-------------|
| **First Law** | Planets orbit in ellipses, not circles |
| **Second Law** | Planets move faster when closer to Sun |
| **Third Law** | Mathematical relationship: orbit time and distance |
| **Significance** | Explained planetary motion mathematically |

### Galileo Galilei (1564-1642)

| Contribution | Details |
|--------------|---------|
| **Telescope** | First to use for astronomy (1609) |
| **Discoveries** | Moon's surface rough; Jupiter's moons; Venus's phases; sunspots |
| **Physics** | Laws of motion; falling bodies |
| **Conflict with Church** | Supported Copernicus |
| **Trial** | 1633; forced to recant |

### Galileo and the Church

| Issue | Description |
|-------|-------------|

## ü§¥ Enlightened Absolutism

### What Was It?

| Feature | Description |
|---------|-------------|
| **Definition** | Absolute monarchs using Enlightenment ideas |
| **Goals** | Rational, efficient administration |

|--------|-------------|

| Reform | Description |
|--------|-------------|
| **Corresponded with philosophes** | Voltaire, Diderot |

| **Education** | Promoted |
