## ÔøΩüìã Unit Overview

Political beliefs shape how Americans view government, policy, and society. This unit explores the foundations of American political ideology, how opinions form and change, and how they are measured. Understanding the spectrum of American political thought is essential for making sense of political debates and elections! üó≥Ô∏è

## üìã Public Opinion

### Measuring Opinion

| Method | Description |
|--------|-------------|
| **Polls** | Surveys of population samples |
| **Focus groups** | In-depth discussions |
| **Interviews** | Individual conversations |
| **Elections** | Ultimate expression |

### Scientific Polling

| Concept | Description |
|---------|-------------|
| **Random sampling** | Everyone has equal chance |
| **Representative sample** | Reflects population |
| **Sample size** | Usually 1,000-1,500 |
| **Margin of error** | Typical ¬±3% |
| **Confidence level** | Usually 95% |

### Polling Terms

| Term | Definition |
|------|------------|
| **Universe/Population** | Group being studied |
| **Sample** | Subset surveyed |
| **Random sample** | Equal chance of selection |
| **Stratified sample** | Proportional representation |
| **Sampling error** | Difference from true value |
| **Margin of error** | Range of uncertainty |

### Potential Polling Problems

| Problem | Description |
|---------|-------------|
| **Selection bias** | Non-representative sample |
| **Question wording** | Leading or confusing questions |
| **Question order** | Sequence affects answers |
| **Social desirability** | Saying what seems acceptable |
| **Non-response** | Some don't participate |
| **Push polls** | Disguised campaign attacks |

### Exit Polls

| Aspect | Description |
|--------|-------------|
| **Purpose** | Predict/explain election results |
| **Method** | Survey voters leaving polls |
| **Issues** | Early voting; mail-in ballots |

### Tracking Polls

| Aspect | Description |
|--------|-------------|
| **Purpose** | Follow trends over time |
| **Method** | Repeated samples |
| **Use** | Campaign strategy |

## üì∞ Media and Political Information

### Types of Media

| Type | Examples |
|------|----------|
| **Traditional/Legacy** | Network TV; newspapers |
| **Cable news** | CNN, Fox, MSNBC |
| **Digital** | Websites; apps |
| **Social media** | Facebook, Twitter, TikTok |
| **Talk radio** | Political talk shows |

### Media Functions

| Function | Description |
|----------|-------------|
| **Inform** | Provide news |
| **Watchdog** | Hold government accountable |
| **Agenda-setting** | Influence what's discussed |
| **Gatekeeping** | Decide what's newsworthy |

### Media Effects Theories

| Theory | Description |
|--------|-------------|
| **Agenda-setting** | Media influences what people think about |
| **Priming** | Media influences standards of judgment |
| **Framing** | Media influences how issues are understood |

### Agenda-Setting

| Concept | Description |
|---------|-------------|
| **Definition** | Media influence on what people think about |
| **Not** | What people think (direct persuasion) |
| **Example** | Coverage of issue makes it seem important |
| **Effect** | Shapes public priorities |

### Framing

| Concept | Description |
|---------|-------------|
| **Definition** | How issue is presented |
| **Example** | "Tax relief" vs. "tax cuts for rich" |
| **Effect** | Shapes interpretation |
| **Source** | Media; politicians |

### Media Bias

| Type | Description |
|------|-------------|
| **Ideological** | Liberal or conservative slant |
| **Selection bias** | What stories are covered |
| **Presentation bias** | How stories are told |
| **Perception of bias** | Viewers see bias differently |

### Echo Chambers and Filter Bubbles

| Concept | Description |
|---------|-------------|
| **Echo chamber** | Only hearing similar views |
| **Filter bubble** | Algorithms limit exposure |
| **Polarization** | Views become more extreme |
| **Effect** | Less understanding of other views |
