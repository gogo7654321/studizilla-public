## üìä Population Growth

### Exponential Growth

| Concept | Description |
|---------|-------------|
| **Definition** | J-shaped curve |
| **Condition** | Unlimited resources |
| **Formula** | dN/dt = rN |
| **r** | Intrinsic growth rate |

### Logistic Growth

| Concept | Description |
|---------|-------------|
| **Definition** | S-shaped curve |
| **Condition** | Limited resources |
| **Formula** | dN/dt = rN √ó (K-N)/K |
| **K** | Carrying capacity |

### Carrying Capacity (K)

| Definition | |
|------------|--|
| **Maximum** | Population that environment can sustain |
| **Limited by** | Food, water, space, mates |
| **Changes with** | Environmental conditions |

### Growth Rate

| When | Growth Rate |
|------|-------------|
| **N << K** | Near maximum (r) |
| **N = K/2** | Maximum (fastest growth) |
| **N = K** | Zero (at capacity) |
| **N > K** | Negative (population declines) |

## ü¶é Reproductive Strategies

### r-Selected Species

| Characteristic | Description |
|----------------|-------------|
| **Many offspring** | Hundreds to thousands |
| **Little parental care** | No investment |
| **Short lifespan** | Quick reproduction |
| **Early maturity** | Reproduce young |
| **Examples** | Insects, rodents, weeds |

### K-Selected Species

| Characteristic | Description |
|----------------|-------------|
| **Few offspring** | 1-2 at a time |
| **Much parental care** | High investment |
| **Long lifespan** | Slow reproduction |
| **Late maturity** | Reproduce older |
| **Examples** | Elephants, whales, humans |

### Comparison

| Factor | r-Selected | K-Selected |
|--------|------------|------------|
| **Offspring** | Many | Few |
| **Size** | Small | Large |
| **Care** | None | Extensive |
| **Environment** | Unstable | Stable |
| **Population** | Below K | Near K |

### FRQ Practice 1: Population Dynamics

**Scenario:** A population of deer on an island has the following characteristics:
- Initial population: 50 individuals
- Birth rate: 0.4 per individual per year
- Death rate: 0.1 per individual per year
- Carrying capacity: 500

**(a)** Calculate the intrinsic growth rate (r) and the population after one year using exponential growth. (3 points)

**(b)** Explain why logistic growth is more realistic than exponential growth for this population. (2 points)

**(c)** At what population size would this population grow fastest? Explain why. (2 points)

**(d)** Describe two density-dependent factors that might limit this deer population. (2 points)

### Sample Answer Rubric:

| Part | Key Points |
|------|------------|
| **(a)** | r = 0.4 - 0.1 = 0.3; N = 50 √ó e^(0.3√ó1) = 50 √ó 1.35 = **67.5 ‚âà 68 deer** |
| **(b)** | Resources are limited; competition increases; carrying capacity exists |
| **(c)** | At N = K/2 = 250; maximum growth rate occurs at half carrying capacity |
| **(d)** | Competition for food; disease transmission; predation pressure |
