## üéµ 4.1 Interval Basics

### Interval Definition

```
INTERVAL:
Distance between two pitches

MELODIC INTERVAL:
Notes played in succession
(One after another)

HARMONIC INTERVAL:
Notes played simultaneously
(At the same time)

NAMING INTERVALS:
Quantity (number) + Quality (type)
Example: Major 3rd, Perfect 5th
```

### Interval Quantity (Number)

```
COUNTING INTERVAL SIZE:

Include both starting and ending notes!

C to D = 2nd (C-D: count 1-2)
C to E = 3rd (C-D-E: count 1-2-3)
C to F = 4th (C-D-E-F: count 1-2-3-4)
C to G = 5th
C to A = 6th
C to B = 7th
C to C = Octave (8th)

UNISON: Same pitch = 1st (or unison)
```

### Simple vs. Compound Intervals

```
SIMPLE INTERVALS:
Octave or smaller (1-8)

COMPOUND INTERVALS:
Larger than an octave

9th = Octave + 2nd
10th = Octave + 3rd
11th = Octave + 4th
12th = Octave + 5th

To reduce compound to simple:
Subtract 7 (or bring down an octave)
9th - 7 = 2nd
```

## üîç 4.2 Interval Quality

### Perfect Intervals

```
PERFECT INTERVALS:
1 (unison), 4, 5, 8 (octave)

These intervals are "perfect" in major scale:

C to C = Perfect unison (P1)
C to F = Perfect 4th (P4)
C to G = Perfect 5th (P5)
C to C = Perfect 8th (P8)

WHY "PERFECT"?
- Most consonant
- Same in major AND minor
- Frequency ratios are simple
```

### Major and Minor Intervals

```
MAJOR INTERVALS:
2nd, 3rd, 6th, 7th in major scale

C to D = Major 2nd (M2)
C to E = Major 3rd (M3)
C to A = Major 6th (M6)
C to B = Major 7th (M7)

MINOR INTERVALS:
One half step smaller than major

C to D‚ô≠ = minor 2nd (m2)
C to E‚ô≠ = minor 3rd (m3)
C to A‚ô≠ = minor 6th (m6)
C to B‚ô≠ = minor 7th (m7)
```

### Augmented and Diminished

```
AUGMENTED (+):
One half step LARGER than perfect or major

P5 (C-G) + half step = A5 (C-G‚ôØ)
M3 (C-E) + half step = A3 (C-E‚ôØ)

TRITONE:
A4 = d5 (same sound, 3 whole steps)
```

## üìè 4.3 Interval Identification Methods

### Half-Step Counting Method

```
COUNT HALF STEPS:

m2 = 1 half step    (C-D‚ô≠)
M2 = 2 half steps   (C-D)
m3 = 3 half steps   (C-E‚ô≠)
M3 = 4 half steps   (C-E)
P4 = 5 half steps   (C-F)
A4/d5 = 6 half steps (tritone)
P5 = 7 half steps   (C-G)
m6 = 8 half steps   (C-A‚ô≠)
M6 = 9 half steps   (C-A)
m7 = 10 half steps  (C-B‚ô≠)
M7 = 11 half steps  (C-B)
P8 = 12 half steps  (C-C)
```

### Major Scale Method

```
USING MAJOR SCALE OF BOTTOM NOTE:

1. Build major scale from bottom note
2. Find top note in that scale
3. If IN scale: M2, M3, P4, P5, M6, M7
4. If not in scale: adjust quality

EXAMPLE: E to C
E major scale: E F‚ôØ G‚ôØ A B C‚ôØ D‚ôØ
C is NOT in E major (C‚ôØ is)
C is one half step LOWER than C‚ôØ
M6 lowered by half step = m6

E to C = minor 6th
```

### White Key Intervals

```
WHITE KEY REFERENCE:

All P1, P4, P5, P8 are PERFECT except:
- B to F = d5 (or F to B = A4)

All 2nds are MAJOR except:
- E-F and B-C = m2

All 3rds are MAJOR:
- C-E, F-A, G-B
Or MINOR:
- D-F, E-G, A-C, B-D

Memorize the exceptions!
```

## üîÑ 4.4 Interval Inversion

### Inversion Concept

```
INTERVAL INVERSION:
Move the bottom note up an octave
(Or top note down an octave)

C-E (3rd) inverted = E-C (6th)

```

### Inversion Rules

```
QUANTITY INVERSION:
Interval + Inversion = 9

2nd ‚Üî 7th  (2+7=9)
3rd ‚Üî 6th  (3+6=9)
4th ‚Üî 5th  (4+5=9)
Unison ‚Üî Octave (1+8=9)

QUALITY INVERSION:
Major ‚Üî minor
Perfect ‚Üî Perfect
Augmented ‚Üî diminished

EXAMPLES:
M3 inverts to m6
P4 inverts to P5
A5 inverts to d4
m2 inverts to M7
```

### Using Inversion for Identification

```
INVERSION SHORTCUT:

If interval is large, invert it!

C to B = ? (large interval)
Invert: B to C = m2 (easy to hear)
m2 inverts to M7
Therefore: C to B = M7

Smaller intervals are easier to identify
```

## üéº 4.5 Consonance and Dissonance

### Consonant Intervals

```
PERFECT CONSONANCES:
Most stable
- Perfect unison (P1)
- Perfect octave (P8)
- Perfect 5th (P5)

IMPERFECT CONSONANCES:
Pleasant but less stable
- Major 3rd (M3)
- minor 3rd (m3)
- Major 6th (M6)
- minor 6th (m6)
```

### Dissonant Intervals

```
DISSONANT INTERVALS:
Create tension, need resolution

MILD DISSONANCE:
- Perfect 4th (in certain contexts)
- Major 2nd (M2)
- minor 7th (m7)

STRONG DISSONANCE:
- minor 2nd (m2)
- Major 7th (M7)
- Tritone (A4/d5)

NOTE: P4 is contextual!
Consonant in chords
Dissonant against bass
```

## üéπ 4.6 Triad Construction

### Triad Definition

```
TRIAD:
Three-note chord built in 3rds

ROOT: Bottom note (names the chord)
THIRD: Note a 3rd above root
FIFTH: Note a 5th above root

C MAJOR TRIAD:
    ‚óè E (5th)
    ‚óè G (3rd)
    ‚óè C (Root)

```

### Triad Qualities

```
FOUR TRIAD QUALITIES:

MAJOR (M):
Root + M3 + P5
C-E-G (C major)

MINOR (m):
Root + m3 + P5
C-E‚ô≠-G (C minor)

DIMINISHED (¬∞):

C-E‚ô≠-G‚ô≠ (C diminished)

C-E-G‚ôØ (C augmented)
```

### Triad Quality Formulas

```

         3rd  +  3rd  =  5th
Major:   M3   +  m3   =  P5
minor:   m3   +  M3   =  P5
dim:     m3   +  m3   =  d5

Major: "big-small" (M3+m3)

```


## üîÑ 4.7 Triad Inversions

### Root Position and Inversions

```
TRIAD POSITIONS:

ROOT POSITION: Root on bottom
    G
    E
    C     (C-E-G, C major)

FIRST INVERSION: 3rd on bottom
    C
    G
    E     (E-G-C)

SECOND INVERSION: 5th on bottom
    E
    C
    G     (G-C-E)
```

### Figured Bass Notation

```
FIGURED BASS SYMBOLS:

ROOT POSITION: 5/3 (or just chord letter)
    C or C5/3

FIRST INVERSION: 6/3 (or just 6)
    C6 or C6/3

SECOND INVERSION: 6/4
    C6/4

EXAMPLE:
C (root position)
C6 (E in bass)
C6/4 (G in bass)
```

### Identifying Inversions

```
FINDING THE ROOT:

Method: Rearrange to stacked 3rds

Given: E-G-C
Rearrange: C-E-G (3rd-3rd)
Root = C (bottom of stacked thirds)
Inversion = 1st (E was in bass)

Given: G-C-E
Rearrange: C-E-G
Root = C
Inversion = 2nd (G was in bass)
```

## üéì Key Takeaways

```
1. INTERVALS HAVE TWO PARTS
   - Quantity (number)
   - Quality (type)
   - Both required for full ID

2. PERFECT INTERVALS ARE SPECIAL
   - Unison, 4th, 5th, octave
   - Can be P, A, or d
   - Not major or minor

3. INVERSION FLIPS QUALITY
   - Major ‚Üî minor
   - Augmented ‚Üî diminished
   - Perfect stays perfect

4. TRIADS BUILD ON THIRDS
   - Stacked 3rds from root
   - Four qualities (M, m, dim, Aug)
   - Three inversions possible

5. DIATONIC TRIADS FOLLOW PATTERN
   - Major: M m m M M m dim
   - Minor: m dim M m m M M
   - Roman numerals show function
```
