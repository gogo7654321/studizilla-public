## ğŸ­ Production in the Short Run

### Time Periods

| Period | Description |
|--------|-------------|
| **Short run** | At least one input is fixed |
| **Long run** | All inputs are variable |
| **Fixed input** | Usually capital (plant, equipment) |
| **Variable input** | Usually labor |

### Production Function

| Concept | Description |
|---------|-------------|
| **Definition** | Relationship between inputs and output |
| **Total product (TP)** | Total output |
| **Marginal product (MP)** | Additional output from one more unit of input |
| **Average product (AP)** | Output per unit of input |

### Formulas

| Measure | Formula |
|---------|---------|
| **Marginal Product** | Î”TP / Î”L |
| **Average Product** | TP / L |

### Law of Diminishing Marginal Returns

| Concept | Description |
|---------|-------------|
| **Definition** | As more of a variable input is added, MP eventually falls |
| **When** | Holding other inputs fixed |
| **Why** | Fixed input becomes scarce |
| **Key insight** | MP still positive, just declining |

### Production Relationships

| Relationship | Meaning |
|--------------|---------|
| **MP > AP** | AP is rising |
| **MP < AP** | AP is falling |
| **MP = AP** | AP is at maximum |

### Stages of Production

| Stage | Description |
|-------|-------------|
| **Stage 1** | MP rising, then MP > AP |
| **Stage 2** | MP falling but positive; efficient range |
| **Stage 3** | MP negative; too much labor |

## ğŸ’° Costs in the Short Run

### Types of Costs

| Cost Type | Description |
|-----------|-------------|
| **Explicit costs** | Actual monetary payments |
| **Implicit costs** | Opportunity costs of owned resources |
| **Economic cost** | Explicit + implicit |
| **Accounting cost** | Explicit only |

### Fixed and Variable Costs

| Type | Description |
|------|-------------|
| **Fixed costs (FC)** | Don't change with output |
| **Variable costs (VC)** | Change with output |
| **Total costs (TC)** | FC + VC |

### Cost Formulas

| Measure | Formula |
|---------|---------|
| **Total Cost (TC)** | FC + VC |
| **Average Fixed Cost (AFC)** | FC / Q |
| **Average Variable Cost (AVC)** | VC / Q |
| **Average Total Cost (ATC)** | TC / Q = AFC + AVC |
| **Marginal Cost (MC)** | Î”TC / Î”Q = Î”VC / Î”Q |

### Cost Curve Shapes

| Curve | Shape | Reason |
|-------|-------|--------|
| **AFC** | Falls continuously | Spreading fixed costs |
| **AVC** | U-shaped | Diminishing returns |
| **ATC** | U-shaped | AFC falls, AVC rises |
| **MC** | U-shaped | Diminishing returns |

### Key Relationships

| Relationship | Meaning |
|--------------|---------|
| **MC < ATC** | ATC is falling |
| **MC > ATC** | ATC is rising |
| **MC = ATC** | ATC is at minimum |
| **MC intersects** | ATC and AVC at their minimums |

### Why MC Crosses ATC at Minimum

| Analogy | Explanation |
|---------|-------------|
| **Grade analogy** | If new test (MC) is below average, it pulls average down |
| **Mathematical** | When MC < ATC, adding more lowers average |
| **At crossing** | MC = ATC means no change to average |

## ğŸ¢ Costs in the Long Run

### Long-Run Average Cost (LRAC)

| Concept | Description |
|---------|-------------|
| **All inputs variable** | Can choose optimal plant size |
| **Envelope curve** | Tangent to short-run ATCs |
| **Planning curve** | Shows minimum cost at each output |

### Economies and Diseconomies of Scale

| Concept | Description |
|---------|-------------|
| **Economies of scale** | LRAC falls as output increases |
| **Diseconomies of scale** | LRAC rises as output increases |
| **Constant returns** | LRAC stays constant |

### LRAC Shape

| Range | LRAC | Returns to Scale |
|-------|------|------------------|
| **Small output** | Falling | Economies of scale |
| **Medium output** | Minimum | Constant returns |
| **Large output** | Rising | Diseconomies of scale |

### Causes of Economies of Scale

| Cause | Description |
|-------|-------------|
| **Specialization** | Workers specialize |
| **Bulk buying** | Lower input costs |
| **Technology** | More efficient equipment |
| **Financial** | Better loan terms |

### Causes of Diseconomies of Scale

| Cause | Description |
|-------|-------------|
| **Coordination problems** | Harder to manage |
| **Communication** | Information gets lost |
| **Bureaucracy** | More layers |
| **Worker alienation** | Less motivation |

### Minimum Efficient Scale

| Concept | Description |
|---------|-------------|
| **Definition** | Smallest output where LRAC is minimized |
| **Importance** | Determines industry structure |
| **High MES** | Fewer firms (natural monopoly) |
| **Low MES** | Many firms possible |

## ğŸ“ˆ Profit Maximization

### The Profit Maximization Rule

| Rule | MR = MC |
|------|---------|
| **If MR > MC** | Produce more (adds to profit) |
| **If MR < MC** | Produce less (reduces profit) |
| **At MR = MC** | Optimal quantity |

### Why MR = MC?

| Logic | Explanation |
|-------|-------------|
| **MR > MC** | Revenue from unit > cost; profit rises |
| **MR < MC** | Cost of unit > revenue; profit falls |
| **MR = MC** | Profit maximized |

### Calculating Profit

| Formula | |
|---------|--|
| **Profit** | TR - TC |
| **Profit** | (P - ATC) Ã— Q |
| **Per unit profit** | P - ATC |

### Profit, Loss, or Break-Even

| Condition | Result |
|-----------|--------|
| **P > ATC** | Economic profit |
| **P = ATC** | Break-even (zero economic profit) |
| **P < ATC** | Economic loss |

## â° Long-Run Equilibrium

### Long-Run Adjustment

| If | Then | Until |
|----|------|-------|
| **Profit > 0** | New firms enter | P falls to ATC |
| **Profit < 0** | Firms exit | P rises to ATC |
| **Profit = 0** | No change | Equilibrium |

### Long-Run Equilibrium Conditions

| Condition | Description |
|-----------|-------------|
| **P = MC** | Profit maximization |
| **P = ATC** | Zero economic profit |
| **P = minimum LRAC** | Productive efficiency |
| **No entry or exit** | Stable |

### Why Zero Economic Profit?

| Reason | Explanation |
|--------|-------------|
| **Free entry** | Profits attract entrants |
| **Free exit** | Losses cause exits |
| **Equilibrium** | No incentive to change |
| **Still earning** | Normal profit (accounting) |
