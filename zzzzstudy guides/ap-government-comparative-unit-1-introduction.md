## ğŸ“š Comparative Politics: Fundamentals

### What Is Comparative Politics?

| Aspect | Description |
|--------|-------------|
| **Definition** | Study of politics by comparing systems |
| **Purpose** | Understand patterns; test theories |
| **Scope** | Domestic politics of different countries |
| **vs. International Relations** | Focus on internal politics |

### Why Compare?

| Reason | Benefit |
|--------|---------|
| **Identify patterns** | What's common across systems |
| **Test theories** | Do explanations hold across cases |
| **Avoid parochialism** | See beyond one's own system |
| **Policy learning** | What works elsewhere |
| **Prediction** | Anticipate changes |

### Methods of Comparison

| Method | Description |
|--------|-------------|
| **Most Similar Systems** | Compare similar countries; find differences |
| **Most Different Systems** | Compare different countries; find similarities |
| **Case study** | In-depth analysis of one country |
| **Large-N** | Statistical analysis of many countries |

### Comparative Concepts

| Concept | Definition |
|---------|------------|
| **State** | Political entity with territory; population; sovereignty |
| **Nation** | Group with shared identity |
| **Nation-state** | State boundaries = nation |
| **Regime** | System of government |
| **Government** | People in power at a time |

## ğŸ›ï¸ State and Regime Types

### State Characteristics

| Feature | Description |
|---------|-------------|
| **Sovereignty** | Supreme authority in territory |
| **Territory** | Defined boundaries |
| **Population** | Citizens |
| **Government** | Institutions of rule |
| **Recognition** | Accepted by other states |

### Regime Types

| Type | Characteristics |
|------|-----------------|
| **Democracy** | Citizen participation; competition; rights |
| **Authoritarian** | Limited participation; controlled competition |
| **Totalitarian** | Complete control; ideology; mobilization |
| **Theocracy** | Religious rule |
| **Hybrid** | Mix of democratic and authoritarian |

### Democratic Regimes

| Feature | Description |
|---------|-------------|
| **Free elections** | Competitive; fair |
| **Civil liberties** | Protected rights |
| **Rule of law** | Laws apply to all |
| **Accountability** | Officials answer to citizens |
| **Pluralism** | Multiple parties; interests |

### Authoritarian Regimes

| Feature | Description |
|---------|-------------|
| **Limited participation** | Controlled or no elections |
| **Restricted rights** | Curtailed freedoms |
| **Centralized power** | Often in leader or party |
| **Limited pluralism** | Controlled opposition |
| **Weak accountability** | Little citizen recourse |

### Classifying Regimes

| Indicator | Democratic | Authoritarian |
|-----------|------------|---------------|
| **Elections** | Free; fair | Controlled; rigged |
| **Competition** | Open | Limited; banned |
| **Participation** | High; protected | Low; controlled |
| **Rights** | Protected | Restricted |
| **Press** | Free | Censored |
