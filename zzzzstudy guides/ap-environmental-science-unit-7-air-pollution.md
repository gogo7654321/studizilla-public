## üå°Ô∏è Temperature Inversions

### Normal Conditions

| Layer | Temperature |
|-------|-------------|
| **Ground** | Warmer |
| **Higher altitude** | Cooler |
| **Result** | Air rises, pollutants disperse |

### Temperature Inversion

| Layer | Temperature |
|-------|-------------|
| **Ground** | Cooler |
| **Higher altitude** | Warmer (inversion layer) |
| **Result** | Air trapped, pollutants concentrated |

### Causes of Inversions

| Cause | Description |
|-------|-------------|
| **Radiation cooling** | Ground cools at night |
| **High pressure** | Descending air warms |
| **Geography** | Valleys trap air |

### Impact on Air Quality

| Effect | Description |
|--------|-------------|
| **Pollutant buildup** | Can't disperse vertically |
| **Smog episodes** | Extended poor air quality |
| **Health effects** | Worse for vulnerable populations |

## üåê Stratospheric Ozone Depletion

### The Ozone Layer

| Location | Stratosphere (15-35 km) |
|----------|-------------------------|
| **Function** | Absorbs UV-B and UV-C |
| **Formula** | O‚ÇÉ |

### Ozone Formation and Destruction

| Natural Cycle | |
|---------------|--|
| **Formation** | O‚ÇÇ + UV ‚Üí 2O; O + O‚ÇÇ ‚Üí O‚ÇÉ |
| **Destruction** | O‚ÇÉ + UV ‚Üí O‚ÇÇ + O |

### Ozone Depleting Substances (ODS)

| Substance | Use |
|-----------|-----|
| **CFCs** | Refrigerants, aerosols, foam |
| **HCFCs** | CFC replacements |
| **Halons** | Fire extinguishers |
| **Methyl bromide** | Pesticide |

### How CFCs Destroy Ozone

| Step | Process |
|------|---------|
| **1** | CFCs rise to stratosphere |
| **2** | UV breaks off Cl atom |
| **3** | Cl + O‚ÇÉ ‚Üí ClO + O‚ÇÇ |
| **4** | ClO + O ‚Üí Cl + O‚ÇÇ |
| **5** | One Cl destroys 100,000 O‚ÇÉ |

### The Ozone Hole

| Location | Antarctica |
|----------|------------|
| **Timing** | September-October (Southern spring) |
| **Cause** | Polar stratospheric clouds + CFCs |
| **Discovery** | 1985 |

### Effects of Ozone Depletion

| Effect | Description |
|--------|-------------|
| **Skin cancer** | More UV reaches surface |
| **Cataracts** | Eye damage |
| **Immune suppression** | Reduced immunity |
| **Phytoplankton** | Reduced ocean productivity |
| **Crop damage** | Some plants sensitive |

### Montreal Protocol (1987)

| Agreement | Description |
|-----------|-------------|
| **Phase out** | CFCs and other ODS |
| **Success** | Ozone layer recovering |
| **Model** | International environmental treaty |
| **Replacements** | HFCs (no chlorine) |
